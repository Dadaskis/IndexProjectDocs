Description

LocationSkyClouds , как и говорится, создаёт облака, которые довольно далеко от игрока.


Их тупо легко использовать: ты просто добавляешь LocationSkyClouds и настраиваешь его параметры по своему вкусу. Нет особых случаев, которые требовали бы какого-либо описания, они просто есть.
