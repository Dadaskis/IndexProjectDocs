Represents a visual task/quest marker in the game world that can be placed at
specific locations. Handles icon display and smooth camera-relative positioning.

Constant

TEX_PATH

"res://modules/task_location_point/assets/textures/"

Base path where marker icon textures are stored

Variable

location_chunk

Undefined/Default

Reference to the location chunk this marker belongs to (for visibility control)

Variable

init_position

Undefined/Default

Original position where the marker was placed

Variable

initialized

false

Flag to track if the marker has been initialized

Variable

force_to_show

false

Whether this marker should always be visible regardless of settings

Variable

ignore_chunks

false

Whether this marker should ignore chunk-based visibility rules

Function

change_icon

tex_filename: String

any

Changes the marker's icon to the specified texture

Function

_physics_process

delta: float

void

Handles smooth positioning relative to camera each physics frame
