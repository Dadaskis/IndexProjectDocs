Description

Проверка демо-статуса, которая убила викторианского ребёнка. Демо-статус в этой игре работает простым способом: Если папка с исполняемым файлом игры имеет файл "demo_status" , это установит демо-статус в true . Этот модуль просто меняет "demo_status" в Values . Однако везде проверяется в основном только DemoStatus.is_demo() . Если "demo_status" был изменён - это не повлияет на статус в DemoStatus . Чтобы полностью переключить статус, используй DemoStatus.switch_demo_status .

Пример использования

if DemoStatus.is_demo():
 # Выполняем демо-логику...
# ...
DemoStatus.switch_demo_status(false) # Мы больше не играем в демо!
