Module: location_chunk

Script: location_chunk

Class Name: LocationChunk

Extends: TriggerBase

CLASS

Page last updated:
November 21, 2025 at 23:07 UTC


LocationChunk class for managing world chunks with occlusion culling
and dynamic loading/unloading of scene elements.
Extends TriggerBase to provide trigger functionality for player detection.
