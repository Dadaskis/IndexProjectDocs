AUTOLOAD

Page last updated:
December 1, 2025 at 21:49 UTC


Autoload singleton for managing global game values that persist
between scenes and checkpoints. Provides simple key-value storage that
automatically saves/loads with the game state.

This kind of global values is different of Values in a way so they are
saved per-save rather than globally across them.

Variable

values

{}

Dictionary storing all global values

any

any
