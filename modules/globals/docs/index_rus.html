Description

Синглтон Globals , который хранит глобальные значения. В отличие от синглтона Values , этот ничего не сохраняет глобально и по сути представляет собой не что иное, как словарь с 2 классными методами set/get сверху. Значения, записанные в этом синглтоне, сохраняются в файле сохранения.

Пример использования

Globals.set_value("lil_something", 1337.0)
Globals.get_value("lil_something", 0.0)
