Description

Globals singleton that stores global values. Unlike Values singleton, this one doesn't save anything globally and basically represents nothing but dictionary with 2 fancy set/get methods on top of it. Values written in this singleton are saved in a save file.

Usage example

Globals.set_value("lil_something", 1337.0)
Globals.get_value("lil_something", 0.0)
