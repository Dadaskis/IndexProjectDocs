Description

Узел Flashbang, который ослепляет NPC и игрока.



Пример использования

var loc_base = LocationBaseScene.get_scene()
var flashbang = Flashbang.new()
loc_base.add_child(flashbang)
flashbang.global_transform.origin = Vector3(1.0, 2.0, 3.0)
flashbang.initialize() # Запускает эффект флешбэнг

Этот модуль НЕ реализует реакцию игрока на флешбэнг и НЕ влияет на поведение NPC. Он содержит только узел Flashbang, который вызывает соответствующие методы у игрока и NPC, поражённых флешбэнгом
