Description

Flashbang node that flashes NPCs and the player as well.



Usage example

var loc_base = LocationBaseScene.get_scene()
var flashbang = Flashbang.new()
loc_base.add_child(flashbang)
flashbang.global_transform.origin = Vector3(1.0, 2.0, 3.0)
flashbang.initialize() # Start flashbang effect

This module is NOT implementing the player's reaction to flashbangs nor affecting NPCs behavior. It has only Flashbang node that calls corresponding methods in player and NPCs that are affected by the flashbang
