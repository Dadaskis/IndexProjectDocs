Module: occlusion_culling

Script: cull_chunk

Class Name: CullChunk

Extends: TriggerBase

CLASS

Page last updated:
October 23, 2025 at 17:51 UTC


Defines a visibility culling volume for occlusion culling system.
Represents a spatial partition that contains meshes and determines
which other chunks are visible from within it. Used by occlusion
culling system to bake data and later optimize the runtime visibility.
