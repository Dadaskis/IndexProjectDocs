Description

LocationZone is some kind of aggressive location optimization. If you enter a different zone, other ones are removing nodes from scene tree, potentially drastically optimizing the game's performance on huge locations. The only problem is that you will have to separate your level into huge parts that are only normally accessible through certain kind of silent teleportation (doors?), can be useful for interiors.

This module is severely outdated and might be a target for future removal. Please consider to use LocationChunk instead.
