Description

Модуль Footsteps. Используется игроком и NPC. Управляет воспроизведением соответствующих звуков шагов в зависимости от поверхности и состояния движения.

Пример использования

# Инициализация
footstep = Footstep.new()
footstep.volume_2D = -30.0
footstep.sound_2D = true # Воспроизводится ли звук с 3D позиционированием или нет
footstep.has_cloth = true
footstep.has_equipment = true
footstep.has_shoes = true
footstep.delay = 0.001
footstep.initialize()

# Использование
footstep.play()

# Дополнительные методы
footstep.play_equipment_sound()
footstep.on_landing()
