Description

This is module is a port of some of the systems from my (Nether) asset streaming project.

The ResourceStreamer node is a universal component that can be used to stream resources to properties of nodes. It features a built-in 3D bounding box that can also be used to automatically load resources based on if the camera is within the bounding box.

The StreamingServer global is a global that is used to asychronously load any resource into memory to be used by ResourceStreamers and other nodes. Loaded resources expire after a set time.
