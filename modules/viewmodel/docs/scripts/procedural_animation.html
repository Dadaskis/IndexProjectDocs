Module: viewmodel

Script: procedural_animation

Class Name: ViewmodelProceduralAnimation

Extends: Object

CLASS

Page last updated:
October 23, 2025 at 21:10 UTC


Handles procedural animations for viewmodels including:
- Weapon bobbing and sway based on movement
- Sight transitions and aiming effects
- Recoil animations
- Environmental interactions (weapon collision)
- Smooth blending between different movement states
