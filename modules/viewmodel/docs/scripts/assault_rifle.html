Animation controller for assault rifle viewmodel
Handles weapon animations, particles, sounds and state management
including shooting, reloading (2 variants), aiming and weapon switching

Initially used for assault rifles, but it can be used for anything that
shares the generic anim set and requires a "full reload" animation

Requires these 5 animations to work properly:
1. "take" (draw weapon)
2. "hide" (holster weapon)
3. "reload" (reload animation)
4. "full reload" (full reload animation when no ammo left)
5. "shoot" (firing animation)
6. "idle" (idle state animation)
