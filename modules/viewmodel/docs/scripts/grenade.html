Animation controller for throwable weapons (grenades, etc.)
Handles weapon states for:
- Drawing/holstering
- Preparing to throw
- Throwing animation
Required animations:
1. "take" - Weapon draw animation
2. "hide" - Weapon holster animation
3. "prepare" - Wind up before throwing
4. "throw" - Throwing animation
5. "hidden" - Empty state animation
