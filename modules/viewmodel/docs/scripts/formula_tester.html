Name

spring_interpolation

Arguments

x: float
target_x: float
spdx: float
resistance: float = 0.2

Return Type

Vector2

Description

Spring interpolation function. Supposed to create spring-like smooth animation, when
the point is going through the target and trying to get to the target again.
It's hard to describe with words, so you better to try that out in formula_tester
x: Current point
target_x: The target point
spdx: Current speed (store it somewhere)
resistance: How resistant the spring?
Returns Vector2.
Vector2.X = New calculated point
Vector2.Y = Your current speed
