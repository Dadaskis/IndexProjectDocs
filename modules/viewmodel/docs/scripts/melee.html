Animation controller for melee/punching weapons
Handles punch animations with screen shake effects
Required animations:
1. "take" - Weapon draw animation
2. "hide" - Weapon holster animation
3. "punch_hit" - Punch animation when hitting target
4. "punch_not_hit" - Punch animation when missing
Uses camera FOV effects during punches
