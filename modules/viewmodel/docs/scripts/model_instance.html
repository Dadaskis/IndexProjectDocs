Module: viewmodel

Script: model_instance

Class Name: ViewmodelModelInstance

Extends: Object

CLASS

Page last updated:
October 23, 2025 at 17:51 UTC


Manages first-person viewmodel (weapon/hands) rendering and behavior including:
- Loading and positioning 3D weapon models
- Handling weapon animations and bone attachments
- Managing particle effects (muzzle flashes, shell casings)
- Controlling viewmodel-specific materials and shaders
- Processing procedural animations and blend states
- Maintaining pools for performance-critical elements
