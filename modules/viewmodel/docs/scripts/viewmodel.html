Module: viewmodel

Script: viewmodel

Class Name: Viewmodel

Extends: Object

CLASS

Page last updated:
October 23, 2025 at 21:10 UTC


Main viewmodel controller that manages first-person weapon rendering and behavior
Handles weapon models, animations, procedural effects, and integrates with:
- Animation system
- Particle effects
- Sound effects
- Procedural animations
Provides interface for gameplay systems to control weapon presentation
