Generic weapon animation controller
Provides base functionality for standard FPS weapons with:
- Draw/hide animations
- Reload animations
- Shoot animations
- Idle states
- Aim down sights functionality
Requires these 5 animations to work properly:
1. "take" (draw weapon)
2. "hide" (holster weapon)
3. "reload" (reload animation)
4. "shoot" (firing animation)
5. "idle" (idle state animation)
