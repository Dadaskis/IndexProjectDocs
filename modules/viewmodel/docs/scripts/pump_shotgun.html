Script: pump_shotgun

Page last updated:
November 17, 2025 at 10:16 UTC


Animation controller for pump-action shotgun
Handles complex reload sequences with:
- Start reload animation
- Loopable shell insertion
- End reload animation
Required animations:
1. "take" - Weapon draw animation
2. "hide" - Weapon holster animation
3. "shoot" - Firing animation
4. "reload_start" - Beginning of reload sequence
5. "reload_loop" - Shell insertion loop
6. "reload_end" - Completing reload sequence
7. "reload_start_full" - Empty weapon reload start
