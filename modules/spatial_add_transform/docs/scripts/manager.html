Module: spatial_add_transform

Script: manager

Class Name: SpatialAddTransformManager

Extends: Node

NODE

Page last updated:
October 23, 2025 at 17:51 UTC


SpatialAddTransformManager manages additional transform modifications that can
be applied to spatial nodes. It allows multiple sources to modify a node's
transform without directly manipulating it, with changes combined each frame.

[]
