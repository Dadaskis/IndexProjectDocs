Module: location_light_volume

Script: location_light_volume

Class Name: LocationLightVolume

Extends: MeshInstance

NODE

Page last updated:
October 23, 2025 at 21:10 UTC


A light volume node for 3D locations that combines
a glowing point light with a volumetric light effect. Handles visual properties like
scaling, coloring, and quality-based visibility, with editor integration for level design.
Used to create atmospheric lighting effects that adapt to camera view angles and distance.
