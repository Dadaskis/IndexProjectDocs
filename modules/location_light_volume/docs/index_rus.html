Description

LocationLightVolume создаёт фейковый объёмный свет, который может быть полезно для улучшения атмосферного точечного освещения в локациях. Довольно понятно, так что... Просто используй его.

Предупреждение: Чрезмерное использование этих узлов может и будет вызывать нагрузку на GPU, что ведёт к проблемам с производительностью. Используй с умом.
