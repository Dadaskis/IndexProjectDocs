Module: damagable

Script: basic_damagable

Class Name: BasicDamagable

Extends: Damagable

CLASS

Page last updated:
October 23, 2025 at 21:10 UTC


A damage handling class for game objects that manages health, destruction effects,
debris generation, and checkpoint saving/loading. Handles damage application, destruction sequences
(including explosions and debris physics), and integrates with the game's save system.

Generally just a basic implementation of Damagable class that is used
by many destroyable elements in this game due to fitting to all needs
