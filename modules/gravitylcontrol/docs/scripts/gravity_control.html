Module: gravity_control

Script: gravity_control

Class Name: GravityControl

Extends: Node

AUTOLOAD

Page last updated:
October 23, 2025 at 17:51 UTC


Autoload singleton for managing global gravity settings
Allows modifying gravity through multiple independent factors that combine multiplicatively
Provides console commands and game systems with ability to adjust gravity without conflicts

Variable

factors

{}

Dictionary tracking all active gravity multipliers by source key

Variable

original_gravity

9.8

Base gravity value (standard Earth gravity)

Variable

gravity

0.0

Current calculated gravity value

Variable

console_gravity

1.0

Special multiplier for console/debug gravity adjustments

Function

update_gravity_param

None

any

Updates the actual physics gravity based on all active multipliers

Function

set_gravity_mult

key: String, mult: float

any

Sets a named gravity multiplier factor
key: Unique identifier for this gravity modifier (e.g. "powerup_slowfall")
mult: Multiplier value (1.0 = normal gravity)

Function

get_gravity_mult

key: String

float

Returns a gravity multiplication value with a provided key
key: Unique identifier for this gravity modifier (e.g. "powerup_slowfall")

Function

set_console_gravity

mult: float

any

Sets the console/debug gravity multiplier
mult: Multiplier value (1.0 = normal gravity)

Function

reset_gravity

None

any

Resets all gravity modifications to default

Function

_ready

None

any

Initialization - sets up default gravity
