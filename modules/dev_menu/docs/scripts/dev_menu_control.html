Module: dev_menu

Script: dev_menu_control

Class Name: null

Extends: Control

NODE

Page last updated:
October 23, 2025 at 21:10 UTC


A comprehensive developer menu system that provides tools for:
- Game debugging and testing features
- Environment/lighting controls
- NPC/prop spawning
- Player loadout management
- Resource editing/overrides
- Sound/music controls
- Achievement unlocking
- Location changing
- And various other development utilities
Acts as an in-game interface for developers to test and modify game systems during development.
This script is the heart of the entire development menu module. It is
working with the UI and is controlling all the logic.
A bit messy, could've been split into several files, but i don't give a fuck.
It was written in 12-hour marathon because Bender wanted this thing so much.
In the end, it helped a lot to have fun with development of this game.
I *always* bind it to F4 key haha
