Module: soundscape

Script: soundscape_class

Class Name: SoundscapeClass

Extends: Resource

RESOURCE

Page last updated:
November 2, 2025 at 9:00 UTC


A resource that defines a soundscape class, includes reverb preset data,
as well as looped and random sounds.

Looped sound: A single looped (prefer .ogg) sample that plays all the time
when this soundscape is active. If multiple looped sounds are defined,
they all will play at the same time, which can be used for a nice
layering effect without any need to go into DAW to make one yourself.

Random sounds: A set of sounds that are randomly playing to make the
soundscape sound more diverse.
