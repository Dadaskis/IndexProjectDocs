Script: st_checkpoint_on_event

Class Name: ST_CheckpointOnEvent

Extends: Node

NODE

Page last updated:
October 23, 2025 at 17:51 UTC


A Script Toolkit node that creates checkpoints when specific game events occur.
Supports delayed saving to avoid issues during frame transitions and can be
configured to create either named or automatic checkpoints.
