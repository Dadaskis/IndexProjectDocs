Script: st_resource_hud_text

Class Name: ST_ResourceHUDText

Extends: Node

NODE

Page last updated:
October 23, 2025 at 17:51 UTC


A Script Toolkit node that displays temporary text on the player's resource HUD
when triggered by a specific location event. Useful for showing pickup messages
or temporary notifications.
