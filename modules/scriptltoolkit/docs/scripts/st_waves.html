Module: script_toolkit

Script: st_waves

Class Name: ST_Waves

Extends: Node

NODE

Page last updated:
October 23, 2025 at 17:51 UTC


A Script Toolkit node that manages wave-based enemy spawning systems.
Handles wave progression, timing, events, messages, and music transitions.
Supports checkpoint saving/loading of wave states.
