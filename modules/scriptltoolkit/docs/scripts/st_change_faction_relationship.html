Script: st_change_faction_relationship

Class Name: ST_ChangeFactionRelationship

Extends: Node

NODE

Page last updated:
October 23, 2025 at 17:51 UTC


A Script Toolkit node that modifies the relationship status
between two factions when triggered by a game event. Enables dynamic faction diplomacy changes
during gameplay, such as making enemies neutral or allies hostile.
