Module: script_toolkit

Script: st_repeater

Class Name: ST_Repeater

Extends: Node

NODE

Page last updated:
October 23, 2025 at 17:51 UTC


A Script Toolkit node that repeatedly fires an event at regular intervals when
activated by a start event, until stopped by a stop event. Supports checkpoint
saving/loading of its state.
