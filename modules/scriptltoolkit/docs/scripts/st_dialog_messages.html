Module: script_toolkit

Script: st_dialog_messages

Class Name: ST_DialogMessages

Extends: Node

NODE

Page last updated:
October 23, 2025 at 17:51 UTC


A Script Toolkit node that handles sequential dialog message display with support for
localization, timed delays between messages, and optional voice lines. Manages
message flow from start to completion with checkpoint saving support.
