Module: script_toolkit

Script: st_use_loadout_item

Class Name: ST_UseLoadoutItem

Extends: Node

NODE

Page last updated:
October 23, 2025 at 17:51 UTC


A Script Toolkit node that manages player equipment loadout through items.
Can remove existing equipment and replace with specified items when triggered by events.
Handles weapons, armor, medicals, special devices and perks with full customization.
