Module: script_toolkit

Script: st_once_only

Class Name: ST_OnceOnly

Extends: Node

NODE

Page last updated:
October 23, 2025 at 17:51 UTC


A Script Toolkit component that ensures an event only triggers successfully once.
Tracks state across game sessions using checkpoints and provides separate success
and failure events based on whether this is the first activation.
