Description

I had a dream of making a game with voicelines until I started working with them and OH GOD how terrible it is to work with them. I would rather have characters mumbling shit instead of them actually speak. Mumbling shit is international too. Lazy? Screw you, mumbling for the win. Either way, voicelines are so deeply tied to this codebase to the point when it would make more sense to keep them. I mean... What if YOU, a poor soul, decided that this game needs voices for some reason? We all are hearing voices in our mind, but you decided that emitting voices by using this game would be even nicer idea? Great. Well, look, you don't really have much to do on this page... Except for some fancy things...

"res://modules/voicelines/assets/samples/" contains all the psychotic voices. It's nicer to organize them by folders, so you can... organize shit, you know? It's nicer to do something like Voicelines.play_voiceline("minc_generic/say_fight0.mp3") rather than Voicelines.play_voiceline("minc_generic_say_fight0.mp3") while dealing with a folder full of trash.
"res://modules/voicelines/assets/samples_voicemess/" has mumbling voices, which work pretty much the same way.

Usage example

var voice = Voicelines.play_voiceline("minc_generic/say_fight0.mp3")
var mumble = Voicelines.play_voicemess("men0.ogg")
var char_voice = Voicelines.play_voiceline_character(
 "minc_generic/say_research0.mp3", character, 20.0, -10.0)
