Function

get_audio

file_name: String

any

Loads an audio sample from the voicelines directory
file_name: Name of audio file to load
Returns: AudioStream or null if not found

Function

play_voiceline

file_name: String

any

Plays a 2D voiceline (no spatial positioning)
file_name: Name of audio file to play
Returns: The created AudioStreamPlayer or null

Function

play_voiceline_character

file_name: String, character: Character, range_add = 0.0, volume_add = 0.0, no_voice_bus = false

any

Plays a 3D voiceline attached to a character
file_name: Name of audio file to play
character: Character to attach sound to
range_add: Additional range for sound attenuation
volume_add: Additional volume adjustment
no_voice_bus: Whether to bypass voice audio bus
Returns: The created AudioStreamPlayer3D or null

Function

get_audio_voicemess

Returns: AudioStream or null if not found

Function

play_voicemess
