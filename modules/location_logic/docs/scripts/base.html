Module: location_logic

Script: base

Class Name: LocationBase

Extends: Spatial

NODE

Page last updated:
October 23, 2025 at 21:10 UTC


LocationBase - Core location management system that handles:
- Location initialization and scene setup
- Player/NPC spawning and management
- Checkpoint saving/loading
- Lighting/environment configuration
- Object pooling and optimization
- Script toolkit integration
- Portal occlusion culling
- Shader compilation
Acts as the central controller for all location-specific logic and systems

All locations must use LocationBase as a scene root
