LocationLogicBase - Base class for location-specific game logic and systems.
Provides core functionality for dialog systems, NPC interactions, event handling,
player management, and location-based gameplay mechanics. Acts as the central
controller for location behaviors, handling everything from character dialogs
to mission tasks and checkpoint systems.
