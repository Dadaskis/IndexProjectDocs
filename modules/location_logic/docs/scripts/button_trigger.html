Module: location_logic

Script: button_trigger

Class Name: LocationButtonTrigger

Extends: TriggerBase

CLASS

Page last updated:
October 23, 2025 at 21:10 UTC


A trigger that creates an interactable button in the game
world. Handles button presses, visual feedback, sound effects, and can be
enabled/disabled. Supports checkpoint saving and editor customization.
