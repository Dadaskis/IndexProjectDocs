Module: location_logic

Script: trigger

Class Name: LocationTrigger

Extends: TriggerBase

CLASS

Page last updated:
December 1, 2025 at 21:49 UTC


A trigger volume that detects body entry/exit and fires
location events. Can be configured for player/NPC-only detection, one-time
triggers, and has checkpoint save/load support. Editor-friendly with automatic
naming and visualization.
