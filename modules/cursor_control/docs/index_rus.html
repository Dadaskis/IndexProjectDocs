Description

Единственная причина существования этого модуля связана с тем, что размер этого проекта стал настолько большим, что использование Input.set_mouse_mode стало сложненьким способом стрелять себе в ноги. Другими словами, Input.set_mouse_mode lacks контроля, когда несколько классов/узлов влияют на видимость курсора, поэтому CursorControl был создан для разрешения этого conflicting поведения, когда курсор мог быть видимыми/невидимыми, когда он не должны быть.

Пример использования

CursorControl.hide("name")    # Скрывает курсор
CursorControl.show("name")   # Показывает курсор
CursorControl.deactivate("name") # Отключает влияние на курсор

Просто как дважды два, однако тебе нужно иметь в виду следующие моменты:
 1. Эта система приоритизирует видимый курсор над невидимым.
 Это значит, что если хотя бы один из предоставленных ключей видимый - курсор тоже станет видимым.
 2. При каждой смене локации вызывается ControlControl.reset() , который сбрасывает влияние всех ключей.
 Просто убедись, что обновляешь ключ своего курсора, если локация изменена.
