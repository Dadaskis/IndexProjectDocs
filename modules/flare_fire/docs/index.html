Description

"Flare Fire" node that creates a weird particle used by chemical and explosion units of M.Inc security. Nobody understood what that is, nobody has to understand what is it either.



Usage example

# Create and configure flare effect
var loc_base = LocationBaseScene.get_scene()
flare_fire = FlareFire.new()
loc_base.add_child(flare_fire)
flare_fire.global_transform.origin = Vector3(1.0, 2.0, 3.0)
flare_fire.start(Color(0.0, 0.5, 0.0)) # Green flare
