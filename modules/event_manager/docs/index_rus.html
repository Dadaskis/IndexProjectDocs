Description

Универсальный менеджер событий, вдохновленный аналогичным из Garry's Mod. Он был довольно полезен в начале проекта, но я перестал использовать его так часто, потому что локационные события покрыли множество случаев. Тем не менее, он всё ещё полезен, поэтому останется в этом проекте надолго.

Пример использования

# Добавить слушатель события
EventManager.add_event_listener(
 "character_killed",
 funcref(self, "on_character_kill")
)

# Запустить событие
EventManager.run_event("character_killed", [victim, killer] )

# Удалить слушатель события
EventManager.remove_event_listener(
 "character_killed",
 funcref(self, "on_character_kill")
)

Используя консольную команду event_manager_list , можно получить список всех доступных событий:

$ event_manager_list
