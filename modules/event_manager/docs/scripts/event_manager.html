A global event bus system for managing and triggering events
Allows decoupled communication between game systems via event listeners

Dictionary storing all event listeners by event name

Dictionary tracking all registered event names

Returns an array of all available event names

Function

event_name: String, func_ref: FuncRef

void

Registers a new event listener for a specific event
event_name: The name of the event to listen for
func_ref: The function reference to call when event triggers

Function

run_event

event_name: String, args = []

void

Triggers an event and calls all registered listeners
event_name: The name of the event to trigger
args: Optional array of arguments to pass to listeners

Function

remove_event_listener

event_name: String, func_ref: FuncRef

void

Removes a specific event listener
event_name: The event name to remove from
func_ref: The specific function reference to remove
