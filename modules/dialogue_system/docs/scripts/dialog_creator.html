Dialogue System Graph Editor - A visual node-based editor for creating and managing dialogue trees.
Allows creating different node types (Start, Answer, Choice, Option, Ending), connecting them,
and saving/loading dialogue structures. Converts the visual graph into a playable dialogue tree resource.
