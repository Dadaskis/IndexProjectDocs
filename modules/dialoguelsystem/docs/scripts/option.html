Module: dialogue_system

Script: option

Class Name: DialogTreeOption

Extends: Resource

RESOURCE

Page last updated:
October 23, 2025 at 17:51 UTC


A resource representing a selectable option in a dialog choice node.
Defines player choices that can either continue the dialog or lead to an ending.

Export

Variable

node_name

Undefined/Default

The name/identifier of this option node in the dialog tree

Export

Variable

is_ending

false

If true, this option leads to a dialog ending rather than continuing

Export

Variable

answer_name

Undefined/Default

Name of the answer node this leads to (when not an ending)

Export

Variable

ending_name

Undefined/Default

Name of the ending node this leads to (when is_ending is true)

Export

Variable

height

Undefined/Default

Visual height/position of this option in UI displays

Export

Variable

text

Undefined/Default

The display text shown to players for this option

Variable

answer

Undefined/Default

Runtime reference to the connected answer node (when not an ending)

Variable

ending

Undefined/Default

Runtime reference to the connected ending node (when is_ending is true)
