Module: trigger_base

Script: trigger_base

Class Name: TriggerBase

Extends: Spatial

AUTOLOAD

Page last updated:
October 23, 2025 at 17:51 UTC


Base class for creating trigger volumes in 3D space with editor visualization
Allows defining rectangular trigger areas with start/end points and handles
collision detection with physics bodies
