Module: post_processing_system

Script: underwater

Class Name: UnderwaterPostProcessing

Extends: PostProcessingBase

RESOURCE

Page last updated:
October 23, 2025 at 21:10 UTC


Simulates underwater visual effects including
refraction, color tinting, blur, and surface distortion. Features adjustable
parameters for water properties and visibility conditions.

Export

Undefined/Default

Texture for water surface distortion/refraction effect
