Script: special_ability_overlay

Class Name: SpecialAbilityOverlayPostProcessing

Handles visual overlay effects for special abilities,
featuring color tinting, mask patterns, and smooth transitions.

Constant

SLOW_RESET_TIME

2.0

Time constant for slow reset of effect

Export

Variable

mask_tex

Undefined/Default

Mask texture for overlay pattern (controls effect visibility)

Variable

shader_mat

Undefined/Default

Shader material implementing the overlay effect

Variable

lerp_speed

5.0

Speed of intensity interpolation (higher = faster transitions)

Variable

intensity

0.0

Current effect intensity (0-1)

Variable

intensity_target

0.0

Target intensity for interpolation

Variable

timer

0.0

Timer for effect duration tracking

Variable

color_current

Color.black

Current displayed color (interpolated)

Variable

color_target

Color.black

Target color for interpolation

Function

custom_initialize

None

any

Initializes the effect by:
1. Loading the special ability overlay shader
2. Applying the mask texture

Function

custom_update

delta: float

any

Updates effect parameters each frame:
1. Interpolates intensity and color values
2. Updates shader parameters
3. Disables effect when fully faded out

Function

start_overlay

color: Color

any

Starts the overlay effect with specified color:
1. Sets target color
2. Begins fade-in animation
3. Resets timing parameters

Function

end_overlay

None

any

Begins fade-out sequence for overlay:
1. Targets zero intensity
2. Fades to black
3. Uses slower transition speed
