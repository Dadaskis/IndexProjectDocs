Module: post_processing_system

Script: night_vision

Class Name: NightVisionPostProcessing

Extends: PostProcessingBase

RESOURCE

Page last updated:
November 4, 2025 at 21:09 UTC


Implements a night vision effect with configurable
mask pattern, noise texture, and tint color. Includes quality level adaptation
and smooth transition when enabled.

Variable

shader_mat

Undefined/Default

The shader material implementing the night vision effect

Export

Variable

nv_mask

Undefined/Default

Mask texture for night vision effect (circular overlay pattern)

Export

Variable

nv_noise

Undefined/Default

Noise texture for night vision effect (static)

Export

Variable

nv_color

Undefined/Default

Tint color for night vision display (typically green)

Variable

intensity

0.0

Current effect intensity (0-1) for smooth transitions

Function

custom_initialize

None

any

Initializes the night vision effect:
1. Loads appropriate shader based on quality settings
2. Applies configured textures and color

Function

on_enable

None

any

Resets intensity when effect is enabled for smooth fade-in

Function

custom_update

delta: float

any

Updates the effect each frame:
1. Smoothly increases intensity when enabled
2. Updates shader parameters
