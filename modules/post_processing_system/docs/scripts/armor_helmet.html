Script: armor_helmet

Class Name: ArmorHelmetPostProcessing

Extends: PostProcessingBase

RESOURCE

Page last updated:
November 21, 2025 at 23:07 UTC


Handles visual effects for armor helmet glass,
including break effects, mask toggling, and pull up/down animations.
Works with checkpoint system for saving/loading state.
