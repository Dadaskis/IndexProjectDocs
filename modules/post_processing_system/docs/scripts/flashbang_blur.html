Module: post_processing_system

Script: flashbang_blur

Class Name: FlashbangBlurPostProcessing

Extends: PostProcessingBase

RESOURCE

Page last updated:
October 23, 2025 at 17:51 UTC


Post-processing effect that simulates a flashbang
explosion visual effect by applying temporary blur and screen offset shaders.
Used for first-person shock/disorientation effects.

Variable

shader_mat

Undefined/Default

ShaderMaterial for the blur effect

Variable

offset

Undefined/Default

ShaderMaterial for the screen offset effect

Function

process_flashbang

None

any

Triggers the flashbang effect sequence:
1. Sets maximum intensity
2. Waits 3 frames
3. Resets intensity to zero

Function

custom_initialize

None

any

Initializes the post-processing effect by loading required shaders
