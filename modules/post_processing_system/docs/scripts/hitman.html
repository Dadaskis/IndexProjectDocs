Module: post_processing_system

Script: hitman

Class Name: HitmanPostProcessing

Extends: PostProcessingBase

RESOURCE

Page last updated:
October 23, 2025 at 17:51 UTC


Post-processing effect that simulates a "hitman" or
assassin-style visual effect (potentially slow motion, color grading, or other
cinematic effects). Manages smooth intensity transitions for the effect.

Variable

shader_mat

Undefined/Default

The ShaderMaterial containing the hitman visual effect

Variable

intensity

0.0

Current intensity of the effect (0-1 range)

Variable

intensity_target

0.0

Target intensity the effect is interpolating toward

Variable

is_active

false

Whether the effect should be active (managed externally)

Function

custom_initialize

None

any

Initializes the effect by loading the hitman shader

Function

custom_update

delta: float

any

Updates the effect each frame, handling smooth intensity transitions
delta: Time since last frame (adjusted for time scale)
