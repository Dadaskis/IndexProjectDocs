Module: post_processing_system

Script: drug_effect

Class Name: DrugEffectPostProcessing

Extends: PostProcessingBase

RESOURCE

Page last updated:
October 23, 2025 at 17:51 UTC


Handles visual effects for drug/impairment states,
including color shifts, distortions, glows and other trippy effects. Manages
smooth transitions between effect states and integrates with checkpoint system.
