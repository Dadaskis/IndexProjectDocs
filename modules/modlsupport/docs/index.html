Description

ModSupport singleton is loading PCK files from "mods/" folder and apply them on top of all the virtual game files including the ones from the main package. Modding in this game didn't really go far beyond that yet, surely you can use IndexMod to produce .PCK files like that. For the time being this module is simple as that, just a loader of these .PCK files that can potentially serve as mods.

Usage Example

ModSupport.load_mods()

That's really it. It is called in "game_start" , so basically all mods are loaded during the initial loading screen.

Keep in mind that due to the fact how ProjectSettings.load_resource_pack works - you can't use mods while playing this game by using editor builds. You can only have mods in an exported game.
