Description


A very simple module that adds UniversalChoiceWindow singleton that allows opening such a fancy cheap-ass window that gives you a unique possibility to turn this shooter into an adventure game. Fuck the healthy understanding of game design, we pour that shit in.

How to use?


Well, you either go the code way or use the ST.

Code



# Array for your pity choices
var array = []
# First choice
var choice0 = UniversalChoiceWindowData.new()
# "Display name" means a normal, human-readable text on the button
choice0.display_name = "Weep"
# "Event name" stands for a location event that'll be emitted when you press that button
choice0.event_name = "weep"
# Another choice just as an example
var choice1 = UniversalChoiceWindowData.new()
choice1.display_name = "Ignore"
choice1.event_name = ""
# You pass that shit 2 arguments - a description text and an array of choices.
UniversalChoiceWindow.open("The air thing here.", [choice0, choice1] )
# TADA you have your damn window open.
