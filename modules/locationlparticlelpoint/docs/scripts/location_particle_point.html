Script: location_particle_point

Class Name: LocationParticlePoint

Extends: MeshInstance

NODE

Page last updated:
October 23, 2025 at 17:51 UTC


This script defines a LocationParticlePoint node.
It is used to place and manage particle effects in a location.
The script provides functionality to initialize, enable, disable, and reinitialize
particle effects, as well as handle quality settings and editor visualization.

Keep in mind this is a simple "particle point", meaning of which it is used
for any kind of particles, but lacks of any control.
If you need to change properties in some particle, you might as well need
to just duplicate the particle file and just change the properties there.
