Module: location_decal_mesh

Script: projectdecal

Class Name: null

Extends: ImmediateGeometry

NODE

Page last updated:

Geometry projection system that creates a projected mesh from overlapping
geometry within an area. Handles clipping, normal calculations, and
UV mapping to create accurate projections of 3D surfaces.
Main logic behind the mesh-based location decals
