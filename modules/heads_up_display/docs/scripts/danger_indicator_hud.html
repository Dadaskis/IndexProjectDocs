Module: heads_up_display

Script: danger_indicator_hud

Class Name: DangerIndicatorHUD

Extends: Control

NODE

Page last updated:
October 23, 2025 at 21:10 UTC


Displays directional indicators for various threats
(damage sources, grenades, suppression) around the screen edges.
Helps player locate off-screen dangers through visual cues.

Undefined/Default

Reference to the rotator control that manages indicator positioning

Undefined/Default

Prefab for damage source indicators

Variable

Undefined/Default
