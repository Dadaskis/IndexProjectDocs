ENG RUS

Description

Расширяемая реализация HUD (Heads-Up Display), которую можно использовать практически с любой камерой, но на данный момент она используется игроком для отображения информации, такой как здоровье, прицел, патроны, задания и т.д.

Пример использования

# Инициализация
var heads_up_display = HeadsUpDisplay.new()
heads_up_display.initialize(self)

# Добавить HUD здоровья, имеющий класс HealthHUD
var health_hud = heads_up_display.load_element("health") as HealthHUD
health_hud.initialize()
health_hud.set_health(100)

# Удалить HUD здоровья
heads_up_display.remove_element("health")

# Скрыть все элементы
heads_up_display.hide_all()

# Показать все элементы
heads_up_display.show_all()

Если тебе нужно больше информации о конкретном элементе HUD - проверь документацию нужного класса HUD

Создание своего собственного элемента HUD


Все элементы хранятся в "res://modules/heads_up_display/assets/elements/" , они следуют шаблону:

"Имя папки элемента"
└── element.tscn
В этом случае "Имя папки элемента" - это имя твоей папки в "elements" , оно может быть любым, просто имей в виду: если ты назовёшь её "ammo" - ты будешь обращаться к ней как "ammo" . Твоя папка элемента может содержать любые файлы в любом порядке, как ты пожелаешь организовать. Это хорошая практика - хранить свои скрипты в папке "/src" и называть их соответственно твоему HUD, например "ammo_hud.gd" . Более того, настоятельно рекомендуется использовать class_name для определения класса твоего HUD, например AmmoHUD , чтобы в будущем ты мог работать с его методами
