Bullet data config. Sets up the bullet's damage, armor damage, force, distance, etc.

Export

Variable

min_damage

1.0

Minimum health damage.

Export

Variable

max_damage

10.0

Maximum health damage.

Export

Variable

min_armor_damage

1.0

Minimum armor damage.

Export

Variable

max_armor_damage

1.0

Maximum armor damage.

Export

Variable

bullet_force

1.0

Bullet's force that is applied on various rigidbodies.

Export

Variable

max_distance

100.0

Maximum distance.
Note: It is still damaging targets even after this distance point.
But it is dealing much less damage as result.

Export

Variable

spread

0.0

Spread value. Be careful, it has a very extreme curve.
Values like 0.2 or 0.3 have sensitive spread.
Try using values near 0.1.
