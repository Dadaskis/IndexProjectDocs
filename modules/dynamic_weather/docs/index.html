Description




"Dynamic" weather. Yeah... Look, I know it sounds weird, but I wanted to get some kind of unified values of Environment and DirectionalLight that could be used across many locations. More than that? A possibility to switch those presets as you go in locations. Sadly it is too hard to implement seamless switching of those as you simply play this game, so it is only recommended to switch those presets with DynamicWeatherManager.update_locbase_environment() only during switches of locations or teleports.

It is also recommended to take a look at:
ST_DynamicWeatherAssign
ST_DynamicWeatherRandomize
ST_DynamicWeatherSet
COM_DynamicWeatherNightOnly

Assign ST helps to call DynamicWeatherManager.update_locbase_environment() without any need to write code for that. In other words, it is simply changing the values of Environment , DirectionalLight and LocationSkyClouds .

Randomize ST - randomizes the time of day and preset ID of current "dynamic" weather.

Set ST - sets your fancy, preferred time of day or preset ID.

"Night Only" Component - helps you to make only those things visible that you want at night. This is helpful to make only certain lights visible at evening/night.
