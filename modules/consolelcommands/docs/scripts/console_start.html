A comprehensive console command script for debugging and game management.
Provides developer commands for gameplay manipulation, debugging tools, system controls,
and game state management. Handles player actions, NPC control, location changes,
visual debugging, achievements, and various game systems through console input.
Also manages console UI state and registers all available commands at startup.
