Description


This module implements "Glitch Camera Scenery" . I don't know how to describe it, but shortly - it's about showing a bunch of point cameras and then switching to pathway camera that goes through some path... Alright this ain't nice way to describe things.

Usage


0. Create a GlitchCameraScenery node
1. Create a bunch of point cameras, set them up
2. Create a bunch of pathway cameras, set them up
3. Pick a nice name for activation event
4. Activate the event whenever you want
5. TADA
