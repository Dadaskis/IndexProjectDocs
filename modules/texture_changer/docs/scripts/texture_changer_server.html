Module: texture_changer

Script: texture_changer_server

Class Name: TextureChangerServer

Extends: Node

AUTOLOAD

Page last updated:
October 23, 2025 at 17:51 UTC


Manages dynamic texture quality adjustments for different material types
(location, viewmodel, character). Processes texture resizing in batches
with quality levels controlled by game settings. Operates across multiple
frames to maintain performance.
