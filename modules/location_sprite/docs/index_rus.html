Description

LocationSprite - это узел, полезный для создания дополнительных эффектов освещения.



Эффект "Glow" позволяет получить более гладкий вид, однако это происходит за счёт дополнительного использования CPU для рейкастов. Чрезмерное использование спрайтов с включённым "Glow" может привести к снижению производительности. Пожалуйста, используй спрайты с этим свойством осторожно.
