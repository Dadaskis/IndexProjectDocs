Module: location_sprite

Script: location_sprite

Class Name: LocationSprite

Extends: MeshInstance

NODE

Page last updated:
November 17, 2025 at 10:16 UTC


A node that manages the appearance and behavior of sprite-like objects.

ENUM

BlendType

Enum to define the blend type for the sprite.

Constant

SPRITES_PATH

"res://modules/location_sprite/assets/sprites/"

Path to the directory containing sprite assets.

Export

Variable

sprite_name

"lens_flare0.tga"

Name of the sprite texture file.

Export

Variable

color

Color.white

Color of the sprite.

Export

Variable

glow

true

Boolean to enable or disable the glow effect.
The glow effect essentially makes it appear nicely on screen,
as if it is a bloom-like lens effect.
WARNING: Uses raycasts, might be a little expensive. Use cautiously.

Export

Variable

scale_xy

1.0

Overall scale of the sprite in both X and Y directions.

Export

Variable

scale_x

1.0

Scale of the sprite in the X direction.

Export

Variable

scale_y

1.0

Scale of the sprite in the Y direction.

Export

Variable

blend_type

BlendType.ADD

Blend type for the sprite.

Export

Variable

glow_game_init

true

Boolean to control the initialization of the glow effect in the game.

Export

Variable

disable_updates

false

Boolean to disable physics process updates.

Export

Variable

keep_on_low_quality

false

Boolean to keep the sprite visible even on low quality settings.

Variable

glow_active

false

Flag to track the active state of the glow effect.
