Module: loadout_system

Script: loadout_presentation

Class Name: LoadoutPresentation

Extends: Control

NODE

Page last updated:
November 23, 2025 at 11:45 UTC


A UI system for mission briefing sequences that handles
page-based presentation flow with subtitles, voiceovers, music, and transition
effects. Manages a sequence of information pages with audio/visual elements.
