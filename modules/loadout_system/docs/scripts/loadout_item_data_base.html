Class Name: LoadoutItemDataBase

RESOURCE

Page last updated:
November 21, 2025 at 23:07 UTC


Base resource class for all loadout items containing
common properties like unlock conditions, crafting requirements, descriptions,
and UI display settings. Used as parent class for specific loadout items.
