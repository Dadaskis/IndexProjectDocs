Module: loadout_system

Script: loadout_system

Class Name: LoadoutSystem

Extends: Node

AUTOLOAD

Page last updated:
November 21, 2025 at 23:07 UTC


Core system for managing player loadouts, progression and resources.
Handles:
- Equipment selection and unlocks
- Player stats/perks progression
- Resource management (crafting materials)
- Loadout application to player character
- Saving/loading loadout state
- Perk activation and effects
- Experience and level progression
