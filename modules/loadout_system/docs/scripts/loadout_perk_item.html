Module: loadout_system

Script: loadout_perk_item

Class Name: LoadoutPerkItem

Extends: LoadoutItemDataBase

RESOURCE

Page last updated:
October 23, 2025 at 17:51 UTC


Defines player perk items that modify gameplay stats through
multipliers, additive bonuses, and boolean flags. Handles perk stacking logic
and provides formatted description text for UI display.
