Page last updated:

Base class for any scripts that are implementing custom loadout item logic.

Create new scripts here:
res://modules/loadout_system/src/logic/

This script allows to run custom logic if speciifc loadout item is selected.
For example, if "perk_engineer" is chosen, "perk_engineer.gd" will be used
to run the custom loadout logic.

Each loadout item is using their own script that must to extend this class.
