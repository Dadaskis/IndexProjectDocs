Description

You wanna some control? I mean more control. I mean a possibility to control what people see in front of their eyes all the time. The window. Well, the name of this module explains it well. Window Control. Power. What else do you want to know?

Usage example

Values.set_value("window_fullscreen", false)
Values.set_value("window_resolution", "640x480") # God's house is 640x480
Values.set_value("display", 0) # Use the first display
