Description

Хочешь немного контроля? В смысле, больше контроля. В смысле, возможность контролировать то, что люди видят перед своими глазами всё время. Окно. Ну, название этого модуля хорошо всё объясняет. Window Control (Управление Окном). Власть. Что ещё ты хочешь знать?

Пример использования

Values.set_value("window_fullscreen", false)
Values.set_value("window_resolution", "640x480") # Дом Бога это 640x480
Values.set_value("display", 0) # Использовать первый дисплей
