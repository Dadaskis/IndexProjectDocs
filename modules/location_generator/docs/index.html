Description

Legacy module in which I've attempted to generate a level with tiles. It wasn't very successful attempt and I'm still not sure why I didn't remove this module, but it is still here.

Main purpose of this module is about "generating" levels using a set of "tiles" that can be connected together into somewhat plausible piece of shit generated location. In other words - I wasn't really content with how it was going on and it left me behind with a strong feeling that I'd do a better job just making these tiles and connecting them all together by myself :)

I wish I'd elaborate more on how it actually works, but you should just take a look at the source code. I don't really think there's any need to elaborate on its functionality as it is incomplete, broke, homeless and doing morphine.

Obsolete module. Might be a target for removal.
