Resource containing generation parameters for procedural
location prefabs. Defines how prefabs are placed during level generation.

Export

Variable

cells_size

Vector3(1.0, 1.0, 1.0)

Size of the prefab in grid cells (width, height, depth)

Export

Variable

random_rotation

true

Whether to apply random rotation when placing this prefab

Export

Variable

cell_snap

1

Grid snapping behavior (1 = snap to whole cells)

Export

Variable

chance

100

Spawn chance percentage (100 = always spawn when eligible)
