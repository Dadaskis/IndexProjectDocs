Data container for location generation cell information.
Stores occupation status, prefab data, and rotation information for procedural
level generation cells.

Variable

is_occupied

false

Flag indicating whether this cell is occupied by any object

Variable

is_prefab

false

Flag indicating if this cell contains a prefab instance

Variable

prefab_name

""

Name/identifier of the prefab if this cell contains one

Variable

rotation_type

0

Type of rotation applied to the cell's contents (0 = no rotation)
