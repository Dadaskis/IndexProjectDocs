Class Name: LocationGenerator

Extends: Spatial

NODE

Page last updated:

Procedural location generator that creates environments from prefab components
using cell-based placement rules. Handles prefab loading, rotation, and border walls.

This shit is broken and i didn't complete it, so...
It's been for about 2 years (from this point) ever since i worked on it.
The idea seemed nice, but then i decided to prioritize working on these
manually.
