Script: com_location_env_blend

Class Name: COM_LocationEnvBlend

Extends: Node

NODE

Page last updated:
November 2, 2025 at 9:00 UTC


A component for smoothly blending environment settings (lighting, fog, etc.)
between different states. Handles gradual transitions of WorldEnvironment properties,
lightmap energy, and individual light parameters when triggered by game events.
Attached to LocationBase.
