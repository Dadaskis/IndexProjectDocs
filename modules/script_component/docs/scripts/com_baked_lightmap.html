NODE

Page last updated:
November 4, 2025 at 21:09 UTC


A comprehensive lightmap baking tool for Godot 3.x
Provides editor tools for baking, adjusting, and managing lightmaps with:
- Automated UV2 unwrapping
- Quality presets and bake configuration
- Lightmap adjustment tools (brightness/contrast/saturation)
- Light baking mode management (realtime/baked/indirect)
- Scene preparation and optimization for baking
- Resource streaming integration
Designed for use with LocationBase scenes and chunked environments

It was made because BakedLightmap sucks
