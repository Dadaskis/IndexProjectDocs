Script: com_moving_sound

Class Name: COM_MovingSound

Extends: Node

NODE

Page last updated:
November 17, 2025 at 10:16 UTC


A component that manages movement-based sound effects for 3D objects.
Plays transition sounds when movement starts/stops and maintains a looping sound
that fades in/out based on movement state. Designed for mechanical/moving objects.
