Script: com_omnilight_optimizer

Class Name: COM_OmnilightOptimizer

Extends: Light

CLASS

Page last updated:
November 2, 2025 at 9:00 UTC


A dynamic light optimization component that automatically
manages OmniLight visibility and shadow rendering based on distance from the
main camera. Helps improve performance by disabling distant lights and shadows.
