Script: com_prop_door_rotating

Class Name: COM_PropDoorRotating

Extends: Node

NODE

Page last updated:
October 23, 2025 at 21:10 UTC


A component that creates interactive rotating doors in 3D space.
Handles door rotation around specified axis, sound effects, locking mechanisms,
player/NPC interaction, and integrates with the game's event and save systems.
