Script: com_prop_door_moving

Class Name: COM_PropDoorMoving

Extends: Node

NODE

Page last updated:
October 23, 2025 at 21:10 UTC


A component that creates interactive sliding doors in 3D space.
Handles door movement along specified axis, sound effects, player/NPC interaction,
damage states, and integrates with the game's event and save systems.
