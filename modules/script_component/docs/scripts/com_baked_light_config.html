Class Name: COM_BakedLightConfig

Extends: Node

NODE

Page last updated:

A configuration component for baked lighting settings that
controls light properties and quality levels for static lighting scenarios.
Attach to light nodes to customize their baking behavior.

Quality level options for lighting
