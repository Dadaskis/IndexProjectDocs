Module: loadout_system

Script: main_control

Class Name: null

Extends: Node

NODE

Page last updated:
October 23, 2025 at 17:51 UTC


Main control of the loadout UI.
Handles the player's equipment selection screen where they can:
- Choose weapons, armor, perks and special devices
- View item stats and requirements
- Craft supplies using resources (disabled)
- Deploy to missions with selected loadout
- Manage skill points and progression
- Display operation briefings and mission info
