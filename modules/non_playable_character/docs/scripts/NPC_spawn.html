Module: non_playable_character

Script: NPC_spawn

Class Name: NPCSpawn

Extends: MeshInstance

NODE

Page last updated:
October 23, 2025 at 17:51 UTC


Spawn point system for NPC characters that:
1. Manages NPC instantiation and respawning
2. Handles spawn conditions (distance, visibility, chance)
3. Supports event-driven spawning
4. Maintains NPC state across checkpoints
5. Provides editor previews for act animations
6. Manages NPC persistence and orphaned instances
Used as both an editor tool and runtime spawn manager
