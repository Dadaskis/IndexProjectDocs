Script: human_weapon_handler

Class Name: NPCHumanWeaponHandler

Extends: Object

CLASS

Page last updated:
October 23, 2025 at 17:51 UTC


Handles weapon functionality for NPC humanoids including shooting mechanics,
burst fire, ammo management, bullet spread calculations, and laser sight.
Integrates with bullet system and NPC character data for combat behavior.
