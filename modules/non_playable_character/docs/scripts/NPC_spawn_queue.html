Script: NPC_spawn_queue

Class Name: NPCSpawnQueue

Extends: Node

AUTOLOAD

Page last updated:
December 1, 2025 at 21:49 UTC


Manages NPC spawning through location events with:
1. Tracking of spawn points by event name
2. Limiting of named NPC spawns
3. Death tracking for spawn limits

Variable

spawns

{}

Dictionary of spawn points keyed by event name (event_name: [spawn_points] )

Variable

named_char_count

{}

Count of currently alive named NPCs (special_name: count)

Function

on_location_change

None

any

Clears all spawn data when location changes

Function

on_death

spec_name: any

any

Handles NPC death events to update spawn counters
spec_name: The special name of the NPC that died

Function

add_spawn

npc_spawn: any, npc: any

any

Registers a new spawn point with the queue
npc_spawn: The spawn point to register
npc: The NPC instance being spawned

Function

on_location_event

event_name: any, args: any

any

Handles location events to trigger NPC spawns
event_name: The name of the triggered event
args: Additional event arguments (unused)

Function

_ready

None

any

Initialize signal connections
