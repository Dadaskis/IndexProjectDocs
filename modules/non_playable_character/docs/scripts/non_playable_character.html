Module: non_playable_character

Script: non_playable_character

Class Name: NonPlayableCharacter

Extends: KinematicBody

NODE

Page last updated:
October 23, 2025 at 17:51 UTC


Base class for all NPC entities that:
1. Manages character data and initialization
2. Handles pathfinding and AI logic
3. Processes physics and gameplay updates
4. Implements combat behaviors
5. Supports special abilities
