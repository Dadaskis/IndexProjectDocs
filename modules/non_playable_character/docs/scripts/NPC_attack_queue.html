Module: non_playable_character

Script: NPC_attack_queue

Class Name: NPCAttackQueue

Extends: Node

AUTOLOAD

Page last updated:
October 23, 2025 at 21:10 UTC


Manages NPC attack actions with rate limiting and queuing
to prevent performance spikes from simultaneous attacks. Handles both gunfire
and grenade attacks with separate cooldown timers.

Constant

SHOOT_WAIT_TIME

0.5

Minimum delay between gunfire attacks (seconds)

Constant

SHOOT_HARD_LIMIT

50

Maximum queued gunfire attacks allowed

Constant

GRENADE_WAIT_TIME

5.0

Cooldown time between grenade throws (seconds)

Variable

shoot_mask

Dictionary tracking which NPCs have pending shots

Variable

shoot_queue

[]

Queue of pending gunfire attacks

Variable

Variable
