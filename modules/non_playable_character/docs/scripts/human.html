Module: non_playable_character

Script: human

Class Name: NPCHuman

Extends: NonPlayableCharacter

CLASS

Page last updated:
November 17, 2025 at 10:16 UTC


A complex NPC controller script that handles humanoid enemy AI behavior including:
- Movement (walking, running, crouching)
- Combat (shooting, grenades, melee)
- Animation and model control
- Pathfinding and navigation
- Damage/health systems
- State management (patrol, combat, etc.)
- Interactions with game world and player
Designed to work with humanoid 3D models and integrate with game systems like
factions, weapons, and physics.
