Script: follow_player

Class Name: _LightUnit_FollowAttack

Extends: NPCLogicStateHuman

CLASS

Page last updated:
November 2, 2025 at 9:00 UTC


NPC logic state for following and attacking behavior:
1. Follows the player while maintaining combat readiness
2. Engages enemies when detected
3. Maintains optimal combat distances
4. Includes grenade throwing logic
5. Handles pathfinding and collision avoidance
