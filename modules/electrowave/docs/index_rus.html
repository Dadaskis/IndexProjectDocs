Description

Узел Electrowave, который может использоваться электро-юнитами M.Inc.

Пример использования

# Получить текущую базовую локацию
var loc_base = LocationBaseScene.get_scene()

# Создать и добавить узел electrowave
var electrowave = Electrowave.new()
loc_base.add_child(electrowave)

# Запустить его
var start_pos = Vector3.ZERO
var target_pos = Vector3.ONE * 10.0
electrowave.start(start_pos, target_pos)

Если хочешь изменить визуал, просто отредактируй "res://modules/electrowave/assets/prefab/electrowave.tscn"
