Description

Limit Data Collector - это модуль, содержащий класс, который собирает данные LocationBase и выводит любые данные, полезные для понимания того, как работать с определёнными "неназванными" ограничениями этой игры.

Пример использования

var loc_base = LocationBaseScene.get_scene()
LimitDataCollector.print_data(loc_base)

Однако настоятельно рекомендуется использовать свойство экспорта в LocationBase под названием show_stats (или "Show Stats" в редакторе)
