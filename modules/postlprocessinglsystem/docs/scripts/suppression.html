Visual effect for suppression/distress states
Features a mask-based overlay that pulses when triggered, with quality-based
shader variants. Used for combat stress or damage feedback.

Constant

SLOW_RESET_TIME

2.0

Time constant for slow reset of suppression effect

Export

Variable

mask_tex

Undefined/Default

Mask texture that defines the suppression effect pattern

Variable

shader_mat

Undefined/Default

The shader material implementing the suppression effect

Variable

lerp_speed

1.0

Speed of effect interpolation (lower = smoother transitions)

Variable

intensity

0.0

Current effect intensity (0-1 range)

Variable

intensity_target

0.0

Target intensity for smooth transitions

Variable

timer

0.0

Timer tracking effect duration

Function

custom_initialize

None

any

Initializes the effect by:
1. Loading appropriate shader based on quality settings
2. Applying the mask texture

Function

custom_update

delta: float

any

Updates the effect each frame:
1. Interpolates intensity toward target
2. Updates shader parameters
3. Disables effect when below visibility threshold

Function

None

Triggers a full-intensity suppression pulse:
1. Immediately sets maximum intensity
2. Enables the effect
