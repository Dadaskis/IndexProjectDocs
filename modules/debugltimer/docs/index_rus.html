Description

Таймер для отладки, который покажет, сколько времени занимает выполнение твоего кода. Не могу сказать, предоставляет ли он точные данные, но он был полезным тем не менее.

Пример использования

DebugTimer.start("name")  # Запускает таймер
DebugTimer.end("name")  # Останавливает таймер
DebugTimer.toggle("name") # "Переключает" таймер

# Насчёт "переключения"...
# Это может быть более полезно для тебя, если твой код становится грязьненьким :)
DebugTimer.toggle("broken_ass_shit")  # Старт
print("blah blah blah")
for iter in range(133722800):     # "Дорогая" операция
 var quick_maths = 1 + 2
DebugTimer.toggle("broken_ass_shit")  # Стоп
