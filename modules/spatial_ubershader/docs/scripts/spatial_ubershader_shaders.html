Shader cache system for SpatialUbershader that manages storage and retrieval
of generated shaders, with editor persistence support.

Constant

SHADER_CACHE_PATH

"res://modules/spatial_ubershader/assets/shader_cache/"

Path where shader cache files are stored

Variable

shaders

{}

Dictionary of cached shaders by their unique keys

Variable

editor_timer

0.0

Timer tracking editor uptime (used for initialization delays)

Variable

has_loaded_shaders

false

Flag indicating if shaders have been loaded from disk

Function

set_shader

key: any, shader: any

any

Stores a shader in cache and optionally saves to disk in editor
key: Unique identifier for the shader
shader: The shader resource to cache

Function

get_shader

key: any

any

Retrieves a shader from cache, loading from disk if needed
key: Unique identifier for the shader
Returns: The cached shader or null if not found

Function

load_shaders

any

Loads all shaders from disk cache (currently disabled)
