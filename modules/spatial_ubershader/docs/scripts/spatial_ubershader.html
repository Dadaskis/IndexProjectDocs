Module: spatial_ubershader

Script: spatial_ubershader

Class Name: SpatialUbershader

Extends: ShaderMaterial

NODE

Page last updated:
October 23, 2025 at 17:51 UTC


A shader material system that combines multiple shader effects into a single
uber-shader, with support for conditional compilation via flags and automatic
shader caching. Manages shader generation from modular effect components.
