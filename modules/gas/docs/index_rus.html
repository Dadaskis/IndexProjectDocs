Description

Узел "Gas", который обычно создается газовыми гранатами или химическими подразделениями M.Inc.

Пример использования

# Создать новый экземпляр эффекта газа
var gas = Gas.new()

# Добавить в иерархию сцены
var loc_base = LocationBaseScene.get_scene()
loc_base.add_child(gas)

# Позиционировать в месте гранаты
gas.global_transform.origin = global_position

# Установить владельца для атрибуции урона
gas.character_owner = character

# Активировать эффект газа
gas.explode()

Визуальные эффекты частиц газа можно редактировать в "res://modules/gas/assets/particles/generic.tscn"
