Weapon class made specifically for "animated items". It can practically be
whatever the hell needs a single "use" animation. However, this class
is a base class with not-so-much functionality.

Weapon system simply switches to an animated item on use, then, during some
animation event the effect of this animated item is applied. When the anim
is over - it is switching back to the earlier gun
