Module: player_weapon_system

Script: weapon_system

Class Name: PlayerWeaponSystem

Extends: Object

CLASS

Page last updated:
October 23, 2025 at 21:10 UTC


Manages the player's weapon inventory and handling, including:
- Loading/storing different weapon types (primary, secondary, melee, grenade, items)
- Weapon switching, reloading, and shooting mechanics
- Animation and viewmodel management
- Ammo pickup and inventory persistence
- Integration with player HUD and input systems
