Class Name: PlayerWeapon

Extends: Resource

RESOURCE

Page last updated:

Base class for all player weapons providing core functionality
Handles weapon animation, viewmodel management, shooting mechanics and HUD updates
Serves as parent class for all weapon types (firearms, melee, grenades, etc)
