Description

Sometimes I feel like I would be a much happier person if I'd have possibility to bend the plane of space and time. Like Max Payne does it. A dude is taking some kind of special painkillers that give him a whole damn possibility to hack the time... But in this codebase, the only thing we can hack is our mind. I mean, not really our mind. We can write less code, alright? We can still write less, we still didn't go too far beyond the point of no return. We can manage the spaghetti... like this module does.

TimeScaleManager gives you a golden opportunity to control the time scale without worrying about other influences. You can register one influence, change it, it will all apply additively, making it possible to control time like you are a fucking Doctor Strange or something like it.

Usage example

var data: = TimeScaleManager.register_data()
data.scale = 0.5
data.audio_scale = 0.5
data.removable = false # Won't be removed on location change... maybe.

TimeScaleManager.remove_data(data)
