Description

MainCamera is a top-notch fucking amazing singleton that does nothing but storing ONE variable. This class is so complex you would NEVER guess that this value is a camera reference! Amazing, right?

Usage example

MainCamera.set_camera(camera) # Sets the camera OMG
var camera = MainCamera.get_camera() # Returns the current main camera

The only reason of its existence is the fact that something has to keep the value of current camera to keep this entire eldritch codebase intact.

I would've needed to write a damn book about this module, but I'm not paid for this shit. Enjoy.
