Script: main_camera

Class Name: MainCamera

Extends: Node

AUTOLOAD

Page last updated:
November 2, 2025 at 9:00 UTC


Global camera manager that tracks and controls the
active camera in the scene. Handles camera switching and notifies other systems
when the active camera changes. Works with LocationBaseScene for debugging.

Signal

on_camera_changed

None

Emitted whenever the active camera changes (new camera or becomes invalid)

Variable

camera

Undefined/Default

Currently active camera reference. Use set_camera() to change it.

Function

set_camera

value: any

any

Sets the new active camera and handles all transition logic
value: Camera node to make active (can be null to clear current camera)

Function

get_camera

None

any

Gets the current active camera with validity checks
Returns: Current camera if valid and in tree, null otherwise
