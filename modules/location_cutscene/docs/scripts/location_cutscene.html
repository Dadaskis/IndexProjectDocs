Module: location_cutscene

Script: location_cutscene

Class Name: LocationCutscene

Extends: Spatial

NODE

Page last updated:
November 17, 2025 at 10:15 UTC


A node for playing a location cutscene. It's... just a cutscene, really.
You can make a cutscene with LocationCutscene.create_cutscene("player_death0")
where "player_death0" is a desired name for a cutscene.
This script should be assigned to a location cutscene prefab.
