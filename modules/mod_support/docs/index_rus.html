Description

Синглтон ModSupport загружает PCK-файлы из папки "mods/" и применяет их поверх всех виртуальных игровых файлов, включая файлы из основного пакета. Моддинг в этой игре пока не зашёл дальше этого, конечно, ты можешь использовать IndexMod для создания .PCK файлов типо таких. На данный момент этот модуль прост, просто загрузчик этих .PCK файлов, которые потенциально могут служить модами.

Пример использования

ModSupport.load_mods()

Это всё. Он вызывается в "game_start" , так что все моды загружаются во время начального экрана загрузки.

Имей в виду, что из-за того, как работает ProjectSettings.load_resource_pack - ты не можешь использовать моды во время игры, используя редакторные сборки. Ты можешь иметь моды только в экспортированной игре.
