Description

Этот модуль является портом некоторых систем из моего проекта (Nether) по потоковой передаче ассетов.

Узел ResourceStreamer - это универсальный компонент, который можно использовать для потоковой передачи ресурсов в свойства узлов. Он имеет встроенный 3D bounding box, который также можно использовать для автоматической загрузки ресурсов в зависимости от нахождения камеры внутри bounding box.

Глобальный объект StreamingServer - это глобальный объект, используемый для асинхронной загрузки любых ресурсов в память для использования ResourceStreamer и другими узлами. Загруженные ресурсы expire по истечении заданного времени.
