Description

Debug timer that would tell you how much does your code take to complete. Cannot tell if it is providing reliable data, but it was useful nonetheless.

Usage Example

DebugTimer.start("name")  # Starts a timer
DebugTimer.end("name")  # Ends a timer
DebugTimer.toggle("name") # "Toggles" a timer

# About "toggling" though...
# It might be more useful for you in case if your code is getting dirty :)
DebugTimer.toggle("broken_ass_shit")  # Start
print("blah blah blah")
