Parts List UI Component

A reusable UI component that displays a scrollable list of character parts
with select/remove functionality. Manages dynamic list creation and cleanup.

Signal

removed_part

part_name

Signal emitted when a part is removed from the list
part_name: The name of the part that was removed

Signal

selected_part

part_name

Signal emitted when a part is selected from the list
part_name: The name of the part that was selected

Function

_ready

None

any

Hides the list and prefab template on ready

Function

remove_part

part_name: any

any

Handles part removal logic and emits signal
part_name: Name of the part to remove

Function

select_part

part_name: any

any

Handles part selection logic and emits signal
part_name: Name of the part to select

Function

clear_list

None

any

Clears all items from the list except the prefab template

Function

update_list

parts: Array

any

Updates the list with new parts data
parts: Array of part names to display
Note: Heavy operation when list is large due to child management

Function

on_new

None

any

Clears the list when creating a new character
