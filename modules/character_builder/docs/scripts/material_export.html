This is a material export node that handles material rendering to textures
for a character builder system. It manages multiple Viewports to generate
texture maps (Diffuse, Normal, MRS, and Emission) from materials
and providing access to the resulting rendered textures.
The system automatically handles material stage detection
(including emission if present) and maintains proper aspect ratios
during rendering.

Essentially this is the entire reason i've made CharacterBuilder, i fucking
hated baking and all that stuff, it's much better this way.
