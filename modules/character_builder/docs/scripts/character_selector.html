Saved Characters Loader UI

This control displays a grid of saved character presets with preview thumbnails.
Handles loading/displaying saved characters and emits signals when selections
are made. Supports live updates when new characters are saved.

Signal

load_character

save_res

Signal emitted when a character is selected for loading
save_res: The loaded Resource containing character data

Constant

SAVE_PATH

"res://modules/character_builder/assets/.saved_characters/"

Path where character saves are stored (.tres files and .png thumbnails)
