Character Builder HitBody Property
Provides selection, transformation, and removal functionality for
character builder body parts.
It acts as an interface between the character builder system
and individual objects, forwarding operations to the real node.

Constant

OBJECT_NAME

"CharacterBuilderBody"

Constant defining the object name for identification

Variable

real_node

Undefined/Default

Reference to the actual 3D node this property controls

Variable

is_selected

false

Selection state flag indicating if this body part is currently selected

Variable

select_filter

false

Filter flag to prevent selection state changes when true

Variable

dont_deselect_others

false

Flag to control whether selecting this should deselect other parts

Variable

can_be_rotated

false

Flag indicating if this body part can be rotated

Variable

rotation_scale

1.0

Multiplier applied to rotation operations

Variable

rotation_angle_offset

Vector3.ZERO

Initial rotation offset applied to this body part

Variable

can_be_positioned

false

Flag indicating if this body part can be repositioned

Variable

position_scale

1.0

Multiplier applied to position operations

Variable

position_angle_offset

Vector3.ZERO

Initial position offset applied to this body part

Variable

can_be_removed

true

Flag indicating if this body part can be removed

Variable

properties

Undefined/Default

Container for additional properties data

Function

select_node

None

any

Selects the associated body part

Marks this body as selected and calls the real node's on_select method if
available. Respects select_filter to prevent unwanted selection changes.

Function

unselect_node

None

any

Deselects the associated body part

Marks this body as unselected and calls the real node's on_unselect method if
available. Respects select_filter to prevent unwanted selection changes.

Function

rotate

axis: Vector3, angle: float

any

Applies rotation to the body part

axis: The axis to rotate around
angle: The rotation angle in radians

Forwards rotation command to real node if can_be_rotated is true and the node
implements on_rotate method.

Function

offset

offset: Vector3

any

Applies position offset to the body part

offset: The translation vector to apply

Forwards position change to real node if can_be_positioned is true and the
node implements on_offset method.

Function

get_position

None

any

Gets the object's world position

Returns: Vector3 - The global position of the real node or Vector3.ZERO if
the node is invalid

Simple getter for the body part's world position.

Function

get_rotation

None

any

Gets the body part's world rotation

Returns: Vector3 - The global rotation of the real node or Vector3.ZERO if
the node is invalid

Simple getter for the body part's world rotation.

Function

remove

None

any

Removes the body part from the scene

Queues the real node for deletion if can_be_removed is true. Does nothing if
the node is already invalid or removal is disabled.
