Character Detail

Creates a customizable character detail part
for a character builder system. It handles loading 3D models, attaching to
skeleton bones, material editing, and transformation controls. The detail
provides control for manipulating character accessories/body parts.

Character Detail objects are supposed to be highly-reusable parts of the character
and as it follows, they are attached to a specific bone and exported appropriately.
It can be anything, really, rig pouches, radios, photos of your mom...
