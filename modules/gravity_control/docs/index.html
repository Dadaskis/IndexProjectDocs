Description

Gravity control of entire LocationBase . If you are using a console command, it is going to keep gravity value across locations.

Usage example

GravityControl.set_gravity("your_name_here", 0.5) # Lowers gravity by half
var gravity = GravityControl.get_gravity("your_name_here") # Returns 0.5
