Description

Fancy demo status checker that killed a victorian child. Demo status in this game works in a simple way: If the game's executable folder has "demo_status" file, that'll set demo status to true . This module simply changes "demo_status" in Values . However, only DemoStatus.is_demo() is checked everywhere for the most part. If "demo_status" have been changed - it won't affect the status in DemoStatus . To switch the status fully, use DemoStatus.switch_demo_status .

Usage Example

if DemoStatus.is_demo():
 # Do the demo logic...
# ...
DemoStatus.switch_demo_status(false) # We aren't playing demo anymore!
