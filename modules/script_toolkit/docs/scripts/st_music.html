Script: st_music

Class Name: ST_Music

Extends: Node

NODE

Page last updated:
October 23, 2025 at 21:10 UTC


A Script Toolkit component for managing music playback through location events.
Handles play/stop commands with both immediate and fade transitions, and supports
music transitions between tracks. Integrates with the Music singleton.
