Module: script_toolkit

Script: st_slow_messages

Class Name: ST_SlowMessages

Extends: Node

NODE

Page last updated:
October 23, 2025 at 21:10 UTC


A Script Toolkit node that displays a sequence of messages with typing animation,
supporting voicelines, player locking, and checkpoint persistence.
It's like a "simple dialog", but a bit different;
Can be used to lock the player entirely.
