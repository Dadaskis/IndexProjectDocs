Module: script_toolkit

Script: st_use_loadout

Class Name: ST_UseLoadout

Extends: Node

NODE

Page last updated:
October 23, 2025 at 17:51 UTC


A Script Toolkit node that manages player loadout equipment. Can give default
weapons/items or restore last used weapons when triggered by events or level start.
Handles weapons, armor, perks and special equipment with configurable options.
