Script: st_NPC_on_trigger_check

Class Name: ST_NPCOnTriggerCheck

Extends: Node

NODE

Page last updated:
November 4, 2025 at 21:09 UTC


A Script Toolkit component that verifies NPC interactions and triggers success events
when specific NPCs activate triggers. Used to create conditional interactions based
on which NPC initiates an event.
