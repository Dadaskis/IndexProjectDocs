Module: script_toolkit

Script: st_explosion_on_event

Class Name: ST_ExplosionOnEvent

Extends: Node

NODE

Page last updated:
November 23, 2025 at 11:46 UTC


A Script Toolkit node that creates explosions in response to game events, with
configurable force, damage, visual effects, and randomized delays. Handles both
physical forces and damage application to nearby objects.

Uses the parent's global position
