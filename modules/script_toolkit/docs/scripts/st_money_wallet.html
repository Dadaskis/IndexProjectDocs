Script: st_money_wallet

Class Name: ST_MoneyWallet

Extends: Node

NODE

Page last updated:
October 23, 2025 at 21:10 UTC


A Script Toolkit component that manages an in-game currency wallet system.
Handles money tracking, saving/loading amounts via checkpoints, and reacting to
location events for checking/resetting/adding money with HUD feedback.
