Script: st_check_enemy_count

Class Name: ST_CheckEnemyCount

Extends: Node

NODE

Page last updated:
October 23, 2025 at 17:51 UTC


A Script Toolkit node that monitors enemy counts and triggers
events when the number of enemies falls below a specified threshold. Useful for
creating progression gates or victory conditions based on enemy elimination.
