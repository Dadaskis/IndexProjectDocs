Script: st_rotate_parent_to_degrees

Class Name: ST_RotateParentToDegrees

Extends: Node

NODE

Page last updated:
November 23, 2025 at 11:46 UTC


A Script Toolkit node that smoothly rotates its parent Spatial node to a target
rotation when triggered by a location event. Uses linear interpolation for
smooth rotation transitions.
