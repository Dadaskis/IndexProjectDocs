Script: st_event_queue

Class Name: ST_EventQueue

Extends: Node

NODE

Page last updated:
October 23, 2025 at 21:10 UTC


A Script Toolkit node that manages a sequential queue of events with
playback control. Supports advancing through events, repeating sequences, and
checkpoint persistence of current position.
