Module: script_toolkit

Script: st_simple_dialog

Class Name: ST_SimpleDialog

Extends: Node

NODE

Page last updated:
November 2, 2025 at 9:00 UTC


A Script Toolkit node that handles simple character dialog sequences with
support for text messages, voicelines, camera effects, and checkpoint saving.
Triggers dialog when specific events occur and manages the dialog state.

Not really a dialog, more like "I talk and you listen"
