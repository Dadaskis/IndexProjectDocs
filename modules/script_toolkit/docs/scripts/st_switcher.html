Module: script_toolkit

Script: st_switcher

Class Name: ST_Switcher

Extends: Node

NODE

Page last updated:
October 23, 2025 at 21:10 UTC


A Script Toolkit node that functions as a stateful switch with multiple control
options. Maintains a boolean value that can be set, toggled, and checked,
triggering different events based on state changes. Persists state across checkpoints.
