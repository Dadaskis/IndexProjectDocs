Script: st_money_trade

Class Name: ST_MoneyTrade

Extends: Node

NODE

Page last updated:

A Script Toolkit node that handles money transactions with success/fail outcomes.
Verifies player has sufficient funds before deducting money and triggers different
events based on the transaction result.
