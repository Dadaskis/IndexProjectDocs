Script: st_wave_data

Class Name: STWaveData

Extends: Resource

RESOURCE

Page last updated:
November 17, 2025 at 10:16 UTC


Resource class that defines configuration data for a single wave in a wave-based system.
Contains timing, events, messages and music transitions for wave start/end.

Export

Variable

enabled

true

Whether this wave is enabled and should be processed

Export

Variable

events

Undefined/Default

List of event names to trigger when this wave starts

Export

Variable

cooldown

2.0

Time delay (in seconds) after wave completion before next wave starts

Export

Variable

begin_message_text

""

Message to display when the wave begins (empty for no message)

Export

Variable

end_message_text

""

Message to display when the wave ends (empty for no message)

Export

Variable

begin_music_name

""

Music track to play when wave begins (empty for no change)

Export

Variable

end_music_name

""

Music track to play when wave ends (empty for no change)
