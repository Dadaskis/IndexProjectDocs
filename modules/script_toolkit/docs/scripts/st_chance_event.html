Script: st_chance_event

Class Name: ST_ChanceEvent

Extends: Node

NODE

Page last updated:
October 23, 2025 at 17:51 UTC


A Script Toolkit node that triggers random events based on probability.
When the check_event occurs, it rolls a random chance to determine whether to trigger
the success_event or fail_event, allowing for randomized gameplay outcomes.
