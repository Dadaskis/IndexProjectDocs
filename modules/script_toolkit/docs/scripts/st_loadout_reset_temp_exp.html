Script: st_loadout_reset_temp_exp

Class Name: ST_LoadoutResetTempExp

Extends: Node

NODE

Page last updated:

A Script Toolkit node that clears all accumulated temporary experience points
when triggered by a location event. This resets temporary progress without
converting it to permanent experience.
