Script: st_NPC_set_walk_target

Class Name: ST_NPCSetWalkTarget

Extends: Node

NODE

Page last updated:
November 17, 2025 at 10:16 UTC


A Script Toolkit component that sets walk targets for NPCs when triggered by location events.
When the specified event occurs, it makes all matching NPCs walk to the position of this node's parent.

Uses parent's global position
