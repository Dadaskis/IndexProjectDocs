Script: st_event_to_one_visible_node

Class Name: ST_EventToOneVisibleNode

Extends: Node

NODE

Page last updated:

A Script Toolkit node that makes only one child node visible
based on received events, hiding all others. Useful for
showing only one object from a group at a time.
