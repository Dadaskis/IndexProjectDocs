Script: st_character_count_message

Class Name: ST_CharacterCountMessage

Extends: Node

NODE

Page last updated:
October 23, 2025 at 21:10 UTC


A Script Toolkit node that displays character count messages in the HUD.
Can show counts of specific NPC types on demand via events.
Supports localization for message text.
