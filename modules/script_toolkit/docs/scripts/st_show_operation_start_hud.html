Script: st_show_operation_start_hud

Class Name: ST_ShowOperationStartHUD

Extends: Node

NODE

Page last updated:
October 23, 2025 at 21:10 UTC


A Script Toolkit node that displays a stylized operation briefing/start screen
on the player's HUD when triggered, typically used for mission introductions
or significant gameplay transitions. Supports optional audio accompaniment.
