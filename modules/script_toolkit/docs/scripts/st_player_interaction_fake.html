NODE

Page last updated:
November 2, 2025 at 9:00 UTC


A Script Toolkit node that simulates player interaction sequences with timed events.
Handles weapon visibility, control locking, sound effects, and optional screen fading
for scripted sequences that don't require actual physical interaction.
