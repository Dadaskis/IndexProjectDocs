Script: st_switch_npc_logic_enabled

Class Name: ST_SwitchNPCLogicEnabled

Extends: Node

NODE

Page last updated:

A Script Toolkit node that enables/disables NPC AI logic for all characters in the scene
when specific events occur. Handles proper reset of NPC states when toggling logic.
