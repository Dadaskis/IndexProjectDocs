Module: script_toolkit

Script: st_task

Class Name: ST_Task

Extends: Node

NODE

Page last updated:
October 23, 2025 at 21:10 UTC


A Script Toolkit node that manages task tracking in the player's HUD.
Supports adding tasks, marking them as partially complete (semicomplete),
and fully completing them through location events.
