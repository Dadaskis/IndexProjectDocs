Script: st_player_set_health

Class Name: ST_PlayerSetHealth

Extends: Node

NODE

Page last updated:
December 1, 2025 at 21:49 UTC


A Script Toolkit node for modifying player health values through location events.
Allows setting both current and maximum health, with optional conditional logic
for maximum health adjustments. Useful for health pickups, damage zones, or
scripted health changes.
