Script: st_change_npc_logic_state_on_trigger

Class Name: ST_ChangeNPCLogicStateOnTrigger

Extends: Node

NODE

Page last updated:
November 4, 2025 at 21:09 UTC


A Script Toolkit node that modifies an NPC's behavior state
when triggered by an event, with optional state blacklisting to prevent certain state changes.
More targeted than ST_ChangeNPCLogicState as it affects only the NPC passed in the event args.
