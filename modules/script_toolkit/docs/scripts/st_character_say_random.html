Script: st_character_say_random

Class Name: ST_CharacterSayRandom

Extends: Node

NODE

Page last updated:
October 23, 2025 at 21:10 UTC


A Script Toolkit node that makes characters speak random messages from a predefined
list when triggered by events. Supports both text display and voice lines, with
optional 3D sound positioning for voice lines.
