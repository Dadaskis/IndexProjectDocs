Script: st_value_check_bool

Class Name: ST_ValueCheckBool

Extends: Node

NODE

Page last updated:
October 23, 2025 at 17:51 UTC


A Script Toolkit node that checks a boolean value from the game's Values system
and triggers different events based on whether the value is true or false.
Useful for creating conditional logic flows in locations.
