Script: st_update_reflection_probe

Class Name: ST_UpdateReflectionProbe

Extends: Node

NODE

Page last updated:
November 21, 2025 at 23:07 UTC


A Script Toolkit node that forces a parent ReflectionProbe to update once
when a specific location event occurs. Includes frame skipping capability
to delay the update for performance optimization.
