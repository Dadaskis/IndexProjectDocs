Script: st_globals_check_bool

Class Name: ST_GlobalsCheckBool

Extends: Node

NODE

Page last updated:
November 10, 2025 at 17:21 UTC


A Script Toolkit node that checks a global boolean value and triggers different
location events based on whether the value is true or false. Useful for
creating conditional event flows in locations based on game state.
