Script: st_battlefield

Class Name: ST_Battlefield

Extends: Node

NODE

Page last updated:
October 23, 2025 at 21:10 UTC


A Script Toolkit node for managing battlefield scenarios with NPC spawns,
kill counters, and event triggers. Handles wave-based combat systems with configurable
start/stop conditions, enemy limits, and victory conditions.
