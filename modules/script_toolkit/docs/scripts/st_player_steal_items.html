Script: st_player_steal_items

Class Name: ST_PlayerStealItems

Extends: Node

NODE

Page last updated:
October 23, 2025 at 21:10 UTC


A Script Toolkit node that manages item theft mechanics in the game. Tracks stolen
items count, enforces limits, and triggers events based on inventory state. Handles
visual feedback and prop removal when items are stolen.
