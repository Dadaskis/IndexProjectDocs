Script: st_player_check_weapons

Class Name: ST_PlayerCheckWeapons

Extends: Node

NODE

Page last updated:
October 23, 2025 at 17:51 UTC


A Script Toolkit node that checks the player's current weapon inventory and
triggers different events based on whether specified weapons are equipped.
Useful for creating weapon-dependent gameplay logic and progression gates.
