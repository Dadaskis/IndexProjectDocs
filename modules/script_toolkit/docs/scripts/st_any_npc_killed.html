Script: st_any_npc_killed

Class Name: ST_AnyNPCKilled

Extends: Node

NODE

Page last updated:
October 23, 2025 at 21:10 UTC


A Script Toolkit node that triggers location events when any NPC
is killed. Used to create gameplay reactions to enemy deaths, like opening doors
or spawning new enemies when certain conditions are met.
