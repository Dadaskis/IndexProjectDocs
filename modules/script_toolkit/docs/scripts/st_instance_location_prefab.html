Script: st_instance_location_prefab

Class Name: ST_InstanceLocationPrefab

Extends: Node

NODE

Page last updated:
October 23, 2025 at 17:51 UTC


A Script Toolkit node that handles instantiating and removing location prefabs
in response to location events. Supports both direct scene references and path-based
loading of prefabs, with checkpoint saving/loading functionality.
