Script: st_hack_waves

Class Name: ST_HackWaves

Extends: Node

NODE

Page last updated:
October 23, 2025 at 21:10 UTC


A Script Toolkit node that implements a simple hacking system with progress tracking.
Manages hacking sequences with distance requirements, progress updates, and failure
conditions. Supports checkpoint saving/loading and integrates with location events.
