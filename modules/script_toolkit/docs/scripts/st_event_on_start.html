Script: st_event_on_start

Class Name: ST_EventOnStart

Extends: Node

NODE

Page last updated:
October 23, 2025 at 21:10 UTC


A Script Toolkit node that triggers events when a location first loads, with support
for checkpoint saving, loading screens, and shader compilation waits. Handles
initialization sequences and can optionally create automatic start saves.
