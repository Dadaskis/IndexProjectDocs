Script: st_change_player_hands

Class Name: ST_ChangePlayerHands

Extends: Node

NODE

Page last updated:
October 23, 2025 at 21:10 UTC


A Script Toolkit node that modifies the player's hand models
when triggered by game events. Allows dynamic swapping of first-person viewmodel hands
during gameplay for cosmetic or gameplay purposes.
