Module: script_toolkit

Script: st_stop_player_and_look_at_parent

Class Name: ST_StopPlayerAndLookAtParent

Extends: Node

NODE

Page last updated:
November 21, 2025 at 23:07 UTC


A Script Toolkit node that temporarily stops player movement and forces their
view to look at this node's parent position. Useful for cinematic moments or
directing player attention to specific objects/events.
