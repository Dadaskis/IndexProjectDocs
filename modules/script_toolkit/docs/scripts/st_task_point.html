Script: st_task_point

Class Name: ST_TaskPoint

Extends: Node

NODE

Page last updated:
November 23, 2025 at 11:46 UTC


A Script Toolkit node that manages task points/markers in the game world.
These points can be added or removed dynamically and support different icons
and display behaviors. Useful for guiding players to objectives.
