Script: st_mission_status_hud

Class Name: ST_MissionStatusHUD

Extends: Node

NODE

Page last updated:
October 23, 2025 at 21:10 UTC


A Script Toolkit node that handles mission status display sequences in the HUD.
Manages cinematic mission complete/fail screens with text displays, sound control,
and player input locking during the sequence.
