Script: st_props_full_list

Class Name: ST_PropsFullList

Extends: Node

NODE

Page last updated:
November 2, 2025 at 9:00 UTC


A Script Toolkit node that manages a list of props that can be swapped in/out
when specific interaction events occur. Tracks progress through the prop list
and triggers completion events.
