Class Name: ST_OneNodeWhitelist

NODE

Page last updated:
October 23, 2025 at 17:51 UTC


A Script Toolkit node that manages scene transitions by listening to location
events. When triggered, it moves a specific child node to the parent and cleans
up remaining nodes. Designed for single-node whitelist functionality.

Constant

SUPPORTED_CLASSES

[]

Array of supported class names that this node can work with
"ALL" means it accepts any node type

Constant

SUPPORTED_SCRIPTS

[]

Array of supported script types (empty means no script restrictions)

Constant

LOC_UNIQUE_VARS

[]

Unique variables that should be unique for this location

Function

on_location_event

loc_event_name: any, args: any

any

Handles location events by moving specified node to parent
loc_event_name: The name of the event/node to process
args: Additional arguments (unused in current implementation)
