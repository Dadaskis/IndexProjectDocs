Module: script_toolkit

Script: st_prop_destroy_on_event

Class Name: ST_PropDestroyOnEvent

Extends: Node

NODE

Page last updated:
November 23, 2025 at 11:46 UTC


A Script Toolkit node that destroys specified props when certain location events occur.
Connects to the location event system and applies massive damage to target prop
when the specified event is triggered, effectively destroying it.
