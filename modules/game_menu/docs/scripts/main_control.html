Module: game_menu

Script: main_control

Class Name: null

Extends: Control

NODE

Page last updated:
October 23, 2025 at 17:51 UTC


GameMenu - Main menu and settings management system for the game. Handles:
- Game settings (graphics, audio, controls)
- Save/load functionality
- Achievements tracking
- Music player (OST)
- Language selection
- Difficulty selection
- News display
Provides UI for starting new games, continuing, and quitting.
Manages transitions between menu states and game states.
