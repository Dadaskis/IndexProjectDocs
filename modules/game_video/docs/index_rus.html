Description

Модуль игрового видео. У меня были планы на него, но он не был таким полезным, потому что видео тяжелые, особенно если вы хотите сохранить хорошее разрешение.

Пример использования

GameVideo.play_video( ["game_start_vid.ogv"] )
# Массив видео будет воспроизводиться одно за другим, позволяя разделить большое видео
yield(GameVideo, "on_finish")
# Продолжить логику после завершения видео
