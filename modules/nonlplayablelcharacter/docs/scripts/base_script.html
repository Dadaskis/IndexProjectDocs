NPC "base scripts" base class. Provides basic API for "base scripts".

Base scripts are supposed to provide additional possibility to enhance
functionality of specific NPC classes without implicitly overriding
the actual NPC class.

Stored at: res://modules/non_playable_character/src/base_scripts/
