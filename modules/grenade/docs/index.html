Description

Grenade node to create grenades. For more configs look at folders of "res://modules/grenade/assets/grenades/"

Usage example"

var loc_base = LocationBaseScene.get_scene()
var grenade = Grenade.new()
loc_base.add_child(grenade)
grenade.character_owner = player.character
grenade.add_collision_exception_with(player) # Prevent self-collision
grenade.initialize("EG-5")

Creating your own grenade


All grenades are stored at "res://modules/grenade/assets/grenades/" , each folder is a unique grenade with the following file structure:

├── logic.tres
└── model.glb
- "logic.tres" is practically any resource that extends functionality of BasicGrenadeLogic (any "grenade logic"). Since this is a resource file - you just configure its properties via exported variables
- "model.glb" is the grenade model that'll be used for visuals
Additional files placed in the folder don't really matter as much, these can be just materials and etc

Write your code and save it at "res://modules/grenade/src/logic/" , you can take a closer look at existing logic to see what they are doing under the hood. Mainly what you can change is the explosion behavior. After that you just can create your own grenade while using your class for "logic.tres" instead.
