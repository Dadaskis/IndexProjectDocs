Description

Узел Grenade для создания гранат. Для дополнительных конфигураций смотрите папки в "res://modules/grenade/assets/grenades/"

Пример использования"

var loc_base = LocationBaseScene.get_scene()
var grenade = Grenade.new()
loc_base.add_child(grenade)
grenade.character_owner = player.character
grenade.add_collision_exception_with(player) # Предотвратить столкновение с собой
grenade.initialize("EG-5")

Создание своей собственной гранаты


Все гранаты хранятся в "res://modules/grenade/assets/grenades/" , каждая папка представляет собой уникальную гранату со следующей структурой файлов:

├── logic.tres
└── model.glb
- "logic.tres" - это практически любой ресурс, расширяющий функциональность BasicGrenadeLogic (любая "логика гранаты"). Поскольку это файл ресурса - вы просто настраиваете его свойства через экспортируемые переменные
- "model.glb" - это модель гранаты, которая будет использоваться для визуализации
Дополнительные файлы, размещённые в папке, не really имеют такого значения, это могут быть просто материалы и т.д.

Напишите свой код и сохраните его в "res://modules/grenade/src/logic/" , вы можете приглядеться к существующей логике, чтобы увидеть, что они делают "под капотом". В основном то, что вы можете изменить, это поведение взрыва. После этого вы просто можете создать свою собственную гранату, используя ваш класс для "logic.tres" instead.
