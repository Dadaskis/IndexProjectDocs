Script: common_editor_hit_body

Class Name: null

CLASS

Page last updated:

A property script for editor objects that provides selection,
transformation (rotation/position), and removal functionality. Acts as an interface
between editor tools and editable objects in the scene.

Used by "Common Editor" module that provides some gizmos that are used by
both ComicsRenderer and CharacterBuilder
