Description

Game start screen that you can see while loading game files. Handles the following cases:
- Resource pack loading
- Mod initialization
- System initialization
- Loading screen animations
- Achievement checks for special conditions



In case if you want to edit the game starting scene, open: "res://locations/scenes/game_start.scn"
