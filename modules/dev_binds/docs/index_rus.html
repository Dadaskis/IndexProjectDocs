Module: dev_binds

ENG RUS

Description

Бинды разработчика на случай, если тебе нужно привязать консольную команду к определённой клавише. Просто говорю, что это работает только для однострочных команд вроде "dev_menu" или "debug_fly", но не для многословных вроде "set_health 100", к сожалению. Используй консольную команду "dev_bind", чтобы увидеть, как это работает.

Пример использования
1. Найди команду, которую хочешь привязать; Например: debug_fly
2. Напиши команду: dev_bind debug_fly
3. Когда попросит нажать клавишу, просто нажми нужную клавишу
4. Та-да

В случае ошибки используй команду dev_bind_remove

Пример использования в коде

# Настройка девинд
DevBinds.setup_bind("debug_fly")
# Удаление девинда
DevBinds.setup_bind_remove()
# Имей в виду: Обе эти функции будут слушать ввод с клавиатуры
# после вызова

# Устанавливает debug_fly на клавишу Z
DevBinds.set_bind(KEY_Z, "debug_fly")
# Удаляет привязки с клавиши Z
DevBinds.set_bind(KEY_Z, "", true)
