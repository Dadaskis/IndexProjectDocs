Description

Developer binds in case if you need to bind console command on certain keyboard key. Just saying it works only for one-word commands like "dev_menu" or "debug_fly" but not for multiword ones like "set_health 100", sadly. Use "dev_bind" console command to see how it works.

Usage example
1. Find a command you want to bind; For example: debug_fly
2. Write a command: dev_bind debug_fly
3. When it'll ask you to press a key, you just press your desired key
4. Tada

In case of mistake, use dev_bind_remove command

Code usage example

# Setup a devbind
DevBinds.setup_bind("debug_fly")
# Remove a devbind
DevBinds.setup_bind_remove()
# Keep in mind: Both of these functions will be listening to keyboard inputs
# after call

# Sets debug_fly on Z key
DevBinds.set_bind(KEY_Z, "debug_fly")
# Removes keybinds from Z key
DevBinds.set_bind(KEY_Z, "", true)
