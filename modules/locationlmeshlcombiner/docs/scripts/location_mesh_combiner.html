Module: location_mesh_combiner

Script: location_mesh_combiner

Class Name: LocationMeshCombiner

Extends: MeshInstance

NODE

Page last updated:
October 23, 2025 at 17:51 UTC


A tool for combining multiple MeshInstance nodes into a single optimized mesh with:
- Automatic LOD material generation
- Collision baking
- Lightmap UV processing
- Material merging
- Editor workflow integration
Handles special cases like glass surfaces and player clips. Used for level optimization.
