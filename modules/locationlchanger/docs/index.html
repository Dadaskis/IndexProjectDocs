ENG RUS

Description

Location Changer helps to change locations with neat loading screen that can be disabled back to generic one depending on an argument provided in change_location

Usage example

# Changes location to a desired location
LocationChanger.change_location("a_prop_polygon")

# Changes location to a desired location without a fancy loading screen
# In this case a "generic" loading screen is used
LocationChanger.change_location("a_prop_polygon", true)

# Reloads current location
LocationChanger.reload_location()


LocationChanger contains features related to preloading, but they aren't fully developed and therefore might as well be obsolete. They are now a target for future removal , so please, keep it in mind.



"res://modules/location_changer/assets/scenes/location_loading.tscn" contains the prefab for loading screens of locations.

Please pay attention
"res://modules/scene_changer/assets/scenes/generic_loading.tscn" is a generic loading screen. Generic loading screen is not a part of this module .
