LocationChunkManager class for managing all LocationChunk instances in a game.
Tracks which nodes have been checked by chunks and maintains a registry of all chunks.
Provides signaling when chunks are entered or exited.

Signal

on_chunk_enter

chunk

Signal emitted when a chunk is entered

Signal

on_chunk_exit

chunk

Signal emitted when a chunk is exited

Variable

nodes_checked

{}

Dictionary tracking which nodes have already been assigned to chunks
Format: {node_path: true}

Variable

chunks

{}

Dictionary of all chunks organized by chunk name
Format: {chunk_name: [chunk1, chunk2, ...] }

Checks if a node has already been assigned to a chunk
