Module: location_changer

Script: location_changer

Class Name: LocationChanger

Extends: Node

AUTOLOAD

Page last updated:

Location changer singleton that handles all level transitions, loading screens,
and preloading of game locations. Manages scene changes while preserving
player state and handling combined location parts.
