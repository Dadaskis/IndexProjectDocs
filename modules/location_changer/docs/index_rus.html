Description

Location Changer помогает переключать локации с аккуратным экраном загрузки, который можно отключить обратно на generic в зависимости от аргумента, предоставленного в change_location

Пример использования

# Переключает локацию на желаемую
LocationChanger.change_location("a_prop_polygon")

# Переключает локацию на желаемую без fancy экрана загрузки
# В этом случае используется "generic" экран загрузки
LocationChanger.change_location("a_prop_polygon", true)

# Перезагружает текущую локацию
LocationChanger.reload_location()


LocationChanger содержит функции, связанные с предзагрузкой, но они не полностью разработаны и поэтому могут быть устаревшими. Они теперь цель для будущего удаления , так что, пожалуйста, имей это в виду.



"res://modules/location_changer/assets/scenes/location_loading.tscn" содержит префаб для экранов загрузки локаций.

Пожалуйста, обрати внимание
"res://modules/scene_changer/assets/scenes/generic_loading.tscn" - это generic экран загрузки. Generic экран загрузки не является частью этого модуля .
