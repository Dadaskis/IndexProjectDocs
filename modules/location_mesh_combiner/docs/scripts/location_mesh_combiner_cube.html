Module: location_mesh_combiner

Script: location_mesh_combiner_cube

Class Name: LocationMeshCombinerCube

Extends: MeshInstance

NODE

Page last updated:
October 23, 2025 at 21:10 UTC


This script defines a LocationMeshCombinerCube node that extends MeshInstance.
It is designed to combine meshes within a specified area, either enclosed or
intersected by the cube. The script provides functionality to filter and
combine meshes based on certain criteria, such as mesh type and dynamic
properties, and visualizes affected areas in the editor.

Basically just a cube that allows you to get all local props/meshes into
a newly made LocationMeshCombiner and etc.
