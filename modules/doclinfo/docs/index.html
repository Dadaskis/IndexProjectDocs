Description

A module for DocInfo resource that makes it possible to write descriptions for whole modules inside of Godot. I mean, you are looking at this description that is written in a DocInfo file! It is used by "ldoclgen" module and if you want to see more usage examples - it is highly advised to check "ldoclgen" description.
