A comprehensive character controlling system for comics rendering

This script provides a complete solution for loading, posing, and managing 3D human models with:
- IK (Inverse Kinematics) system for natural limb positioning
- Weapon attachment and handling
- Animation pose blending
- Bone-level manual editing capabilities
- Comics-style rendering features (rim lighting)
- Save/Load functionality for character states

Key Features:
- Loads human models from predefined paths with configurable options
- Implements IK controls for hands/feet with adjustable targets
- Supports weapon attachments with proper hand positioning
- Provides animation pose presets and blending
- Enables manual bone-level editing in "pose mode"
- Handles character layer visibility for comics rendering
- Manages materials and shaders (especially rim lighting)
- Implements collision shapes for selection/interaction
- Full serialization support for saving/loading character states

Usage:
- Models are loaded from res://modules/human_model/assets/models/
- Weapons from res://modules/human_model/assets/weapons/
- Animations from res://modules/human_model/assets/animation/
- Control through ComicsRendererControl UI integration

Technical Notes:
- Uses SkeletonIK for inverse kinematics
- Maintains separate control nodes for each IK chain
- Handles bone remapping for consistent naming
- Manages material instances for visual customization
- Implements hit detection via custom collision bodies
