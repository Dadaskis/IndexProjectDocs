Script: shader_layers

Class Name: null

Extends: Node

NODE

Page last updated:
October 23, 2025 at 21:10 UTC


Shader layers script for comic rendering pipeline that:
- Manages multiple viewports for different render passes
- Handles texture passing between shaders
- Controls render timing and updates
- Provides final composited output

Simply saying it is used to apply all these fancy glitchy effects on top
of the raw rendered image
