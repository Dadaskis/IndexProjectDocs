Module: comics_renderer

Script: comics_renderer

Class Name: null

Extends: Node

NODE

Page last updated:
November 21, 2025 at 23:07 UTC


Main controller for comics rendering system

Handles scene management, object selection, camera control, and rendering pipeline for:
- Loading/saving comic scenes
- Managing environment and character layers
- Controlling render cameras and lighting
- Processing multi-pass rendering (environment, characters, masks)
- Exporting final comic panels

Provides UI integration through ComicsRendererControl
Works with specialized nodes (characters, props, lights)
Manages viewports for separate rendering passes
