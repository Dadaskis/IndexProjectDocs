Module: comics_renderer

Script: camera_control

Class Name: null

Extends: KinematicBody

NODE

Page last updated:
October 23, 2025 at 21:10 UTC


First-person camera controller for comics renderer with:
- WASD movement with sprint/jump/crouch
- Mouse look with sensitivity/clamping controls
- Toggleable movement (hold middle mouse to move/look)
- Camera environment settings
- Smooth movement interpolation
- Main camera registration system
