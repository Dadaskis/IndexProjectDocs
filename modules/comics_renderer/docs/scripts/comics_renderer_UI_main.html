Module: comics_renderer

Script: comics_renderer_UI_main

Class Name: null

Extends: Node

NODE

Page last updated:
October 23, 2025 at 21:10 UTC


Main controller for comics rendering system - handles UI, scene management, rendering presets,
and post-processing effects for a comic book/storyboard creation tool. Manages:
- Location/scene loading/saving
- Character/prop creation
- Camera controls
- Shader effects and post-processing
- Preset management
- Final image rendering and export
