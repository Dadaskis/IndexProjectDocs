A node giving a possibility to create complex prefabs. Replaces
"--P" in String values to ensure uniqueness.
^^^ Useful for location events in general.
Also supports arguments, you can do things like...
loadout::notices = ["123"]
In this case key will be split by "::" and
first part will be used as a node path, while the second
is used as a value to set. All prefabs can be configurable.
