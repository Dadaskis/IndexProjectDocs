A config for a location prop. Basically, instead of fully relying on
the exported variables, this config was made to make it easier to configure
properties of the props that are placed on the location. For example,
to make explosive barrels explosive all the time, or make the small
briefcases possible to pick up.

There *are* exported vars in LocationProp that allow you more configuration
for the individual props, but most of them are used to "override" values
from here.

All config files are stored here:
res://locations/props/config/

Format: [prop_name] .tres
Many of them are created by LocationProp itself automatically anyway,
whenever you place down a prop on the location.
