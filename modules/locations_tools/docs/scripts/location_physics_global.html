LocationPhysicsGlobal - Provides global utilities for managing physics layers
and collision masks. Used for static body layer assignment and management.

Variable

static_body_counter

0

Counter for cycling through static body layers

Function

get_static_body_layer_bit

None

int

Gets the next available static body layer bit (cycles through layers 10-18)
Returns: The layer bit index to use

Function

assign_static_body_layers

body: any, value = true

any

Sets collision mask bits for all static layers (10-18) on a physics body
body: The physics body to modify
value: Boolean to set all bits to (true = enable, false = disable)

Function

assign_bit_layers

layers: int

int

Sets multiple layer bits at once (layers 10-18)
layers: Initial layer mask to modify
Returns: The modified layer mask with static layers enabled
