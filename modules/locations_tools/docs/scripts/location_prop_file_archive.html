Prop file archive singleton for location props.
Stores a lot of data related to location props to reuse during initialization.
Mainly used for the optimization purposes.

Constant

EXPORTED_PATH

"res://locations/props/exported/"

Path to the GLTF model files of props

Variable

postfixes

[]

A list of possible model file naming postfixes

Variable

props

{}

A dictionary to store file paths related to prop model
Format: "prop_name" -> ["prop_name.glb", "prop_name_PHYS.glb", ...]

Variable

mod_times

{}

Modification times of the props, not sure why it exists,
but i wouldn't dare to touch it.

Function

reset_data

None

any

Resets all data, that is props and mod_times

Function

update_archive

None

any

Updates the props and mod_times to contain the useful data
for the initialization of location props

Function

get_prop_files

prop_name: String

any

Returns the list of files used by a prop

Function

get_mod_time

prop_name: any

any

Get the last modification time of a specified prop
