Module: locations_tools

Script: location_blockout_mesh

Class Name: LocationBlockoutMesh

Extends: MeshInstance

NODE

Page last updated:
November 4, 2025 at 21:09 UTC


This script defines a LocationBlockoutMesh node that severely extends MeshInstance.
It is designed to manage and manipulate mesh instances within a game location,
providing functionality for handling various properties and behaviors of these
meshes, such as collision, damage, and visual effects.
