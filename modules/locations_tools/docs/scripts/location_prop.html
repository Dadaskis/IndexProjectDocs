Module: locations_tools

Script: location_prop

Class Name: LocationProp

Extends: MeshInstance

NODE

Page last updated:
October 23, 2025 at 17:51 UTC


A comprehensive prop system for LocationBase scenes that handles:
- Model loading and management
- Physics setup (static/dynamic/animated)
- Damage system integration
- Pickup/use functionality
- LOD management
- Checkpoint saving/loading
- Material and color customization
- Editor tooling for level design
- And much more; Satisfying you in weird ways
Acts as an ultimate tool for all kind of interactive props in location scenes.
