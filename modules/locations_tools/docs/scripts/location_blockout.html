Script: location_blockout

Class Name: LocationBlockout

Extends: Spatial

NODE

Page last updated:
November 21, 2025 at 23:07 UTC


A LocationBlockout node.
It is designed to manage blockout models in any game location, providing
functionality to load, update, and process blockout models and their
associated meshes, collisions, and other properties.
