Page last updated:

A resource that defines physical and rendering
properties for blockout geometry material.
Used to configure collision, lighting, physics
and damage properties across multiple blockout meshes.

Stored here:
res://locations/blockouts/material_config/

If the first material of blockout mesh surface is the one having a config,
it'll be loaded and all the properties of it will be assigned to the entire
blockout mesh.
