Script: locgen_location_data

Class Name: LocGenLocationData

Extends: Resource

RESOURCE

Page last updated:
October 23, 2025 at 17:51 UTC


Resource containing configuration data for procedural
location generation. Defines prefabs and parameters used when generating
locations in the game world.

Export

Variable

fill_prefabs

[]

Array of prefab names to use when filling empty space in the location

Export

Variable

use_border_wall

true

Whether to generate border walls around the location

Export

Variable

border_wall_prefabs

[]

Array of prefab names to use for border walls

Export

Variable

border_cells_size

2.0

Size/thickness of the border wall area in grid cells

Export

Variable

border_wall_corner_prefab

""

Special prefab to use for border wall corners (if empty, uses regular wall prefab)
