Module: characters

Script: character_damagable

Class Name: CharacterDamagable

Extends: Damagable

CLASS

Page last updated:
October 23, 2025 at 17:51 UTC


Damagable object for characters. Has a plenty of useful methods to work with
character as well as reference to original author and damage multiplier.
Runs a bit of special logic if player is the character that is being damaged.

Constant

HIT_DECAL_TEXTURE

preload()

Hit decal texture. Usually refers to texture of blood.

Constant

HIT_PARTICLE

preload()

Preloaded hit particle scene. Usually refers to blood particles.

Constant

VIOLENT_HIT_PARTICLE

preload()

Different hit particle used for high-damage weapons (example: BSR-10)

Constant

VIOLENT_HIT_MINIMUM_DAMAGE

50

How much damage is required to initiate 'violent' particle

Constant

EXPLOSION_HIT_PARTICLE

preload()

Literally just a particle for "EXPLOSION" damage type

Constant

GIBBED_HIT_PARTICLE

preload()

Scene for overkill (gibbing), BLOOD IS FUEL.

Constant

ARMOR_HIT_PARTICLE

preload()

Preloaded armor hit particle scene that is created
if you damage character that wears armor.

Variable

character

Undefined/Default

Reference to character that this damagable belongs to.

Variable

damage_multiplier

1.0

Overall damage multiplier. Useful for various hitbox multipliers.

Function

create_hit_effect

position: Vector3, direction: Vector3

any

Instances and emits a new hit particle at given position/direction.

Hit particles are created when a character receives not-so-much damage
Check VIOLENT_HIT_MINIMUM_DAMAGE constant, if damage value is below
this constant, this particle will be created

position - global position to create a particle at
direction - global direction vector to "look" at

Function

create_violent_hit_effect

position: Vector3, direction: Vector3

any

Instances and emits a new violent hit particle at given position/direction.

Violent hit particles are the blood particles that are created when
a damager deals a lot of damage to a character.
"A lot of damage" is a value above VIOLENT_HIT_MINIMUM_DAMAGE constant.

position - global position to create a particle at
direction - global direction vector to "look" at

Function

create_explosion_hit_effect

position: Vector3, direction: Vector3

any

Instances and emits a new explosion hit particle at given position/direction.

Explosion hit particles are created when a character is struck by an explosion

position - global position to create a particle at
direction - global direction vector to "look" at

Function

create_gibbed_hit_effect

position: Vector3, direction: Vector3

any

Instances and emits a new gibbed hit particle.

Gibbed hit effect is created when a character is blown up with an explosion.
By "blown up" i mean damaged by an explosion to death.

It might've been funny to make them explode into gibs if they are damaged
too hard.

position - global position to create a particle at (UNUSED, OBSOLETE)
direction - global direction vector to "look" at (UNUSED, OBSOLETE)

Function

create_armor_hit_effect

position: Vector3, direction: Vector3

any

Instances and emits an armor hit particle at given position.

Function

damage_armor

value: float

any

Damage armor of this character. Doesn't take any damage if "can_take_damage()" returns false.

Function

damage

value: float, damage_type: = Damagable.DamageType.GENERIC, position: = Vector3.ZERO, direction: = Vector3.ZERO, author: Character = null

void

Damage health of this character. Doesn't take any damage if "can_take_damage()" returns false.
value - Amount of damage.
damage_type - Damage type.
position - Damage hit global position.
direction - Damage hit global direction.
author - Damage author (reference to another character)
