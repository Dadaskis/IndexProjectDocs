Description


This is it, that's the point when we can turn this game into immersive-sim, oh yeah. Actually, nevermind, this may not happen.

There are 2 ways to use this thing, you either write code or you use ST.

Code


First argument is password (as a string). Second argument is an event name when access is granted.
KeypadMenu.open("1337", "keypad_access_granted")

Script Toolkit



Simply use "st_keypad" . "Open Event" is an event that has to happen when this keypad needs to be opened (shit it's harder to write than I thought). "Password" is the password, keep in mind there's 4 digit limit. "Access Event Name" is an event that happens when you enter the password right.
