DebugDraw - A runtime debugging visualization system that draws lines, shapes, and volumes
Provides immediate-mode rendering of debug geometry like AABBs, lines, planes and triangles
with configurable colors and persistence. Useful for collision debugging, AI visualization,
and other in-game debugging needs.

Variable

render

Undefined/Default

Reference to the ImmediateGeometry node used for rendering

Variable

timer

0.0

Timer for controlling render updates

Variable

delay

0.1

Delay between render updates (in seconds)

Variable

remember_lines

[]

Array of lines that should persist between frames

Function

_ready

None

any

Initializes the debug drawing system

Function

render_line

pos0: Vector3, pos1: Vector3, color: Color, remember: = false

void

Draws a line between two points in 3D space
pos0: Starting position of the line
pos1: Ending position of the line
color: Color of the line
remember: If true, the line will persist between frames

Function

render_aabb

aabb: AABB, color: Color, remember = false

void

Renders an Axis-Aligned Bounding Box (AABB) with colored lines
aabb: The AABB to render (contains position and size)
color: Color to use for the box lines
remember: If true, the box lines will persist between frames

Function

render_plane

v0: Vector3, v1: Vector3, v2: Vector3, v3: Vector3, color: Color

void

Renders a quadrilateral plane by drawing it as two triangles
v0, v1, v2, v3: The four corner vertices of the plane in 3D space
color: The color to use for the plane

Function

render_triangle

v0: Vector3, v1: Vector3, v2: Vector3, color: Color

void

Renders a single triangle
v0, v1, v2: The three vertices of the triangle in 3D space
color: The color to use for the triangle

Function

_physics_process

delta: any

any

Physics process that handles periodic rendering updates
delta: Time elapsed since last frame
