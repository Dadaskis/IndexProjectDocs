Handles all communication between UI controls and text box data. Listens for:
- Text changes (updates parent Label)
- Talk variant changes (shows/hides speech bubble styles)

It is basically some kind of editor relay? I don't know, it just works.

Function

on_text_changed

text: any

any

Updates the parent label's text when the text box data changes
text: The new text to display in the parent label

Function

on_talk_variant_changed

value: any

any

Handles talk variant changes by showing/hiding the appropriate child elements
value: The index of the talk variant to display (hides all others)

Function

set_text_box_data

data: ComicsTextBoxData

any

Initializes the text box editor with data and connects signals
data: The ComicsTextBoxData containing the text box properties
