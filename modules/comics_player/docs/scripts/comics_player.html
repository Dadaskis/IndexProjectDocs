Comics Player System - Handles display and animation of comic pages

Key Features:
- Loads and displays comic pages with images, text boxes and effects
- Manages page transitions and animations
- Handles voice lines, sound effects and text animations
- Supports click-to-advance functionality
- Processes different text box types (speech bubbles, action text, etc.)

Usage Flow:
1. Load comics data (play_comics() or play_comics_by_name())
2. Creates pages with images and text boxes (create_page())
3. Animates elements in sequence (process_animation())
4. Handles user input and timing between elements
5. Cleans up when finished

Note: Works with ComicsData, ComicsPageData and ComicsTextBoxData resources
