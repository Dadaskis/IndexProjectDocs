Module: comics_player

Script: comics_editor

Class Name: null

Extends: Node

NODE

Page last updated:
October 23, 2025 at 21:10 UTC


Comics editor system for creating and managing interactive comic pages.
Handles page layout, images, text boxes, audio, and visual effects.
Features include:
- Pattern-based page templates
- Image positioning/scaling
- Text box management with variants
- Audio integration (voice lines/sound effects)
- Save/load functionality for comics projects
Used by ComicsPlayer to display the final comic sequences
