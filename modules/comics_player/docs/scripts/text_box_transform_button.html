Transform Button - A multi-purpose control that handles:
- Position dragging
- Rotation dragging
- Scale dragging

Features:
- Quick reset on short clicks (<0.2s) for rotation/scale
- Live updating of connected text_box_data properties

Works by climbing 4 parents up to find the target Control node,
then modifies its transform properties based on mouse movement.
I know, it's a dirty trick, but i don't really give a fuck for now.
It's gonna be a problem for the future-me, if he's gonna be alive.
