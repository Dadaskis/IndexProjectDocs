Page last updated:

Color grading system that applies LUT (Look-Up Texture) effects to the game.
Handles both runtime and editor usage, with save/load support. Main features:
- Applies different color grading LUT textures to change visual style
- Toggles effect on/off through game Values system
- Auto-initializes game effect instance on _ready
- Supports editor preview of LUT effects
- Maintains state through the saving system
Note: Requires properly formatted LUT textures in the specified LUT_PATH

any

Changes the active Look-Up Texture (LUT) for color grading

- Specified LUT doesn't exist
- Default "none" LUT fails to load
