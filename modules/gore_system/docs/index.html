Description

Gore System written by our fellow Unnamed Entity. In this particular case, the gore system is used for creation of gibs when characters are exploded. Logic related to creation of gibs is called at CharacterDamagable class.

Usage example

var gibscene = GoreSystemGibs.new() # Instance gibbing script
gibscene.translation = Vector3(1.0, 2.0, 3.0) # Set pos
LocationBaseScene.get_scene().add_child(gibscene) # Initialize gibbing
