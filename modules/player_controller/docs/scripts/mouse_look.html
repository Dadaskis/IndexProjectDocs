Module: player_controller

Script: mouse_look

Class Name: PlayerMouseLook

Extends: Object

CLASS

Page last updated:
November 23, 2025 at 11:46 UTC


Controls player camera/view rotation using mouse input.
Handles first-person view controls with sensitivity settings, view clamping,
weapon/hud effects, and special look-at functionality.
