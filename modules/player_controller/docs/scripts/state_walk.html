Module: player_controller

Script: state_walk

Class Name: PlayerStateWalk

Extends: Object

CLASS

Page last updated:
November 10, 2025 at 17:21 UTC


Handles all player movement logic including:
- Basic walking/running with speed variations (normal, sprint, slow)
- Jumping and air movement control
- Crouching with collision checks
- Stair climbing/descending with raycast detection
- Camera bobbing and viewmodel effects
- Fall damage calculation
- Footstep sounds and movement statistics
Integrates with weapon handling and character systems
