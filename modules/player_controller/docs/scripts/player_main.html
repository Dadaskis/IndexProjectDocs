Module: player_controller

Script: player_main

Class Name: PlayerController

Extends: KinematicBody

NODE

Page last updated:
November 21, 2025 at 23:07 UTC


PlayerController - Main player character controller script that handles:
- Movement states (walking, swimming, climbing ladders)
- Combat/weapon handling
- Health/damage systems
- "Inventory/interaction" systems (how else can i describe that?)
- Visual effects (post-processing, camera, animations)
- HUD/UI management
- Save/load functionality
Acts as the central hub for all player-related gameplay systems and mechanics.

It's not like a huge monolithic script, right?.. right?
