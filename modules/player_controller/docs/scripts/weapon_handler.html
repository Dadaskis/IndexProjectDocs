PlayerWeaponHandler - Manages all weapon-related functionality for the player,
acting as an interface between player input/state and the weapon system.
Handles weapon switching, firing, reloading, and movement state propagation.

Variable

Parent node that will hold weapon models in the scene

Variable

Variable
