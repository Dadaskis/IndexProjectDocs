Singleton for tracking water volumes and handling water interactions.
Maintains a list of all water bodies in the scene and provides methods to check
water collisions and trigger water effects like splashes.

Variable

water_list

[]

Array storing all active water volumes in the scene

Function

add_water

water: PlayerControllerWater

any

Adds a water volume to the tracking system
water: The PlayerControllerWater instance to add

Function

reset

None

any

Clears all tracked water volumes (typically called on scene changes)

Function

is_point_in_water

point: Vector3

bool

Checks if a point is inside any water volume
point: The world position to test
Returns: true if the point is in water, false otherwise

Function

make_splash

point: Vector3

any

Triggers splash effects at a point in water
point: The world position where splash should occur
