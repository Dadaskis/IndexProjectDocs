Description

Модуль для ресурса DocInfo , который позволяет писать описания для целых модулей прямо внутри Godot. Вот же, ты сейчас смотришь на это описание, которое написано в файле DocInfo ! Он используется модулем "_doc_gen" , и если хочешь посмотреть больше примеров использования - настоятельно рекомендую заглянуть в описание "_doc_gen" .
