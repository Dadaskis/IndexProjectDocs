Script: location_LOD

Class Name: LocationLOD

Extends: Node

AUTOLOAD

Page last updated:

Location LOD (Level of Detail) system for managing and baking simplified
versions of mesh materials with combined texture atlases.
Handles both editor-time baking and runtime LOD distance calculations.
