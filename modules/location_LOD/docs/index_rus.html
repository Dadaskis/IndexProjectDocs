Description

Класс LocationLOD предоставляет функциональность для "запекания" LOD-модели с одним диффузным материалом, чтобы сохранить количество отрисовок.

Пример использования

LocationLOD.bake_LOD_mesh(sample_mesh_LOD, self.mesh.resource_path)
yield(LocationLOD, "bake_complete")
# Сохраняем результаты
var data = LocationLOD.get_baked_LOD_data()
var mesh_LOD = data.mesh
# ...

Это действительно всё, что тебе нужно знать об этой штуке. Если ты хочешь увидеть её использование в текущих реализациях, проверь LocationProp , LocationBlockoutMesh и LocationMeshCombiner .
