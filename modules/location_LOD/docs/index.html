Description

LocationLOD class provides functionality to "bake" a LOD model with one diffuse material to save draw-calls amount.

Usage example

LocationLOD.bake_LOD_mesh(sample_mesh_LOD, self.mesh.resource_path)
yield(LocationLOD, "bake_complete")
# Store results
var data = LocationLOD.get_baked_LOD_data()
var mesh_LOD = data.mesh
# ...

That's really what you need to know about this thing. If you want to see usage of it in current implementations, check LocationProp , LocationBlockoutMesh and LocationMeshCombiner .
