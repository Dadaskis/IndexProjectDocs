Description

The sole reason of this module's existence is related to the fact that scope of this project have became so big to the point when usage of Input.set_mouse_mode became a sophisticated way of shooting your legs off. In other words, Input.set_mouse_mode lacks of control when multiple classes/nodes are affecting the cursor's visibility, hence why CursorControl was made to resolve this conflicting behavior when cursor could be visible/invisible when it shouldn't be.

Usage Example

CursorControl.hide("name")    # Hides the cursor
CursorControl.show("name")   # Shows the cursor
CursorControl.deactivate("name") # Disables influence on cursor

Simple as that, however, you need to keep in mind the following moments:
 1. This system is prioritizing visible cursor over invisible one.
 That means if at least one of the provided keys are visible - cursor will turn visible too.
 2. Every location change, ControlControl.reset() is called, which resets influence of all keys.
 Just make sure to update your cursor's key if location is changed.
