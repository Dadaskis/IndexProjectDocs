Script: cursor_control

Class Name: CursorControl

Singleton that manages mouse cursor visibility and capture state based on priority keys.
Allows different systems to request cursor visibility while handling focus changes.
Implements a priority system where any active key can make the cursor visible.

I made this thing because using Input.set_mouse_mode turned to hell
in scope of this project. Always something broke, always something
hidden or showed up the cursor when nobody really needed it.

You should use only following functions:
CursorControl.hide("name") # Hide a cursor
CursorControl.show("name") # Show a cursor
CursorControl.deactivate("name") # Disable influence

Keep in mind that CursorControl prioritizes visible cursor, so if one of the
provided IDs require a visible cursor - it'll be visible nonetheless.

It is also getting reset on any location change, keep it in mind.
