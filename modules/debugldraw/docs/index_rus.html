Module: debug_draw

Documentation last edited: October 22, 2025 at 10:46 UTC

ENG RUS

Description

Специальный модуль для отладки, в случае если тебе нужно рисовать некоторые debug линии с помощью immediate geometry узла без особых проблем. Ранее он использовался для тестирования контроллера игрока, но он всё ещё useful для других видов 3D отладки (как дебаггинг Rooms и Portals).

Пример использования

# Рендерит красную линию И запоминает её существование
DebugDraw.render_line(
 Vector3(0.0, 0.0, 0.0),
 Vector3(0.0, 0.0, 1.0),
 Color.red,
 true
)

# Рендерит синий AABB только один раз без запоминания
DebugDraw.render_aabb(
 get_transformed_aabb(),
 Color.blue,
 false
)

# Рендерит красную плоскость один раз в кадре
DebugDraw.render_line(
 Vector3(0.0, 0.0, 0.0),
 Vector3(0.0, 0.0, 1.0),
 Vector3(1.0, 0.0, 0.0),
 Vector3(1.0, 0.0, 1.0),
 Color.red
)

# Рендерит красный треугольник один раз в кадре
DebugDraw.render_triangle(
 Vector3(0.0, 0.0, 0.0),
 Vector3(0.0, 0.0, 1.0),
 Vector3(1.0, 0.0, 0.0),
 Color.red
)
