Script: achievement_unlocked_display

Class Name: AchievementUnlockedDisplay

Extends: Node

Page last updated:

Independent UI system for displaying unlocked achievement notifications.
Features:
- Works even when game is paused
- Plays unlock sound effect
- Shows achievement icon and localized name
- Smooth fade in/out animations
- Automatically localizes text
