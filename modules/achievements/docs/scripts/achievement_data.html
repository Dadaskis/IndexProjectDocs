Module: achievements

Script: achievement_data

Class Name: AchievementData

Extends: Resource

RESOURCE

Page last updated:
October 23, 2025 at 21:10 UTC


A config resource for individual achievements.
All achievements should be stored in: "res://modules/achievements/assets/data/"

Each achievement requires:
1. A .tres resource file defining its properties
2. An optional icon in: "res://modules/achievements/assets/icons/"
(using the same filename as the achievement resource)

Constant

ICONS_PATH

"res://modules/achievements/assets/icons/"

Path to achievement icons directory

Constant

PLACEHOLDER_ICON_PATH

Default icon used when achievement-specific icon isn't found

Constant

Undefined/Default

Default icon used when achievement-specific icon isn't found
