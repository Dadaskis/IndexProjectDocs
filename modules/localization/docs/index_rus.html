Description

Синглтон Localization , который переводит текст в этой игре. Если ты хочешь изменить определённый перевод, загляни в ".game_export/utilities/translation_tool" - это инструмент для изменения переводов в этой игре.

Пример использования

# Получить переведённую версию "Test text"
# "HUD" — это расположение/контекст этого текста, который показывается в утилите перевода
var translation = Localization.get_text("Test text", "HUD")

# Принудительно проверяет "Test text 2", чтобы он был включён в следующий дамп файла локализации
Localization.force_check("Test text 2")

# Проверяет все дочерние узлы данного узла и переводит их, если возможно
Localization.translate_all_nodes(node, "HUD")
