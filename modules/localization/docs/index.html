Description

Localization singleton that translates text in this game. If you want to change certain translation, look up ".game_export/utilities/translation_tool" , this is a tool to change translations in this game.

Usage example

# Get translated version of "Test text"
# "HUD" is a location/context of this text that is shown up at translation utility
var translation = Localization.get_text("Test text", "HUD")

# Forces to check "Test text 2" so it would be included in the next localization file dump
Localization.force_check("Test text 2")

# Checks all child nodes of the given node and translates them if possible
Localization.translate_all_nodes(node, "HUD")
