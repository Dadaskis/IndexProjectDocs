Module: localization

Script: localization

Class Name: Localization

Extends: Node

AUTOLOAD

Page last updated:
October 23, 2025 at 21:10 UTC


Localization system singleton that handles text translation, tracking of
untranslated strings, and automatic dumping of translation files. Supports
various UI elements and provides tools for managing multiple languages.

Constant

DUMP_TIME

20.0

Time interval (in seconds) between automatic dumps of untranslated strings

Variable

original_text_dict

{}

Dictionary mapping original texts to empty strings (for tracking untranslated)

Variable

translations

{}

Dictionary storing all translations (original -> translated)

Variable

text_locations

{}

Dictionary tracking where texts appear in the game (for context)

Variable

text_to_original

{}

Reverse dictionary mapping translations back to originals

Variable

dump_timer

0.0

Timer counting up to DUMP_TIME for automatic file dumping

Variable

needs_dump

false

Flag indicating if we need to dump untranslated strings
