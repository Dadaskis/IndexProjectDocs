Description

Содержит только синглтон DebrisControl , который является глобальной очередью с дополнительными шагами. Он был создан для оптимизации обломков, удаляя их чрезмерное количество. Если общее количество обломков идёт свыше лимита, самые старые куски будут удалены первыми.

Пример использования

if not DebrisControl.can_create_debris(): # Останавливает инициализацию, если слишком много обломков
 return
DebrisControl.add_debris(self)     # Добавляет обломки

Все узлы обломков должны иметь метод remove_debris , который будет вызываться при их "удалении". Сам DebrisControl НЕ удаляет никакие узлы, потому что они могут требовать переиспользования (aka удаление из дерева сцены вместо queue_free их).
